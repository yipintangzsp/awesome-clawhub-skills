# 🚀 6 个自动工作流配置

**张 sir，配置好后自动干活！**

---

## 1️⃣ 公众号工作流

**用途：** 从选题到发布全搞定

**触发命令：**
```
写公众号文章，主题：[主题]
```

**自动执行：**
1. 搜索热点资讯
2. 生成文章大纲
3. 写初稿
4. 建议配图
5. 排版建议

---

## 2️⃣ 朋友圈运营流

**用途：** 自动生成朋友圈内容

**触发命令：**
```
生成今天的朋友圈内容
```

**自动执行：**
- 早上发干货
- 中午发观点
- 晚上发故事

---

## 3️⃣ AI 视频工作流

**用途：** 从 0 到 1 做视频

**触发命令：**
```
做个 AI 视频，主题：[主题]
```

**自动执行：**
1. 搜索热门方向
2. 写 Seedance 提示词
3. 生成图片建议
4. 输出制作流程

---

## 4️⃣ 多平台分发流

**用途：** 一鱼多吃

**触发命令：**
```
把这篇文章分发到多平台
```

**自动执行：**
- 公众号版（2000 字）
- 小红书版（500 字 + 配图）
- 朋友圈版（100 字）
- Twitter 版（280 字）

---

## 5️⃣ 每日简报

**用途：** 每天早上推送

**定时：** 早 8 点

**内容：**
1. 今天天气
2. 日历安排
3. 未读邮件
4. Skill 收入数据
5. 励志语录

---

## 6️⃣ 收入监控

**用途：** 监控 SkillPay 收入

**定时：** 每晚 10 点

**内容：**
- 今日下载量
- 今日收入
- 热门 Skill
- 周/月累计
