## 二、准备工作：三样东西就够了

别被"AI 智能体"吓到，准备过程比你想的简单。

### 1. 运行环境
一台普通电脑就行：
- Windows 10 及以上
- macOS 10.15 及以上  
- Linux 任意发行版

想让它 24 小时在线？租个云服务器，2 核 4G 足够，每月约$2。

### 2. AI 模型 API Key（龙虾的大脑）

这是核心成本，但比你想的便宜。

**新手推荐（国内模型）：**
- **MiniMax**：免费额度多，性价比高，适合入门
- **Kimi**：访问速度快，中文理解好
- **智谱 AI**：稳定，适合长期用

**进阶用户（国际模型）：**
- **Claude**：逻辑推理强，适合复杂任务
- **GPT-4**：综合能力顶级，但成本高

我的建议：先用 MiniMax 或 Kimi 入门，跑通流程后再考虑升级。

### 3. 通讯渠道（指挥龙虾的麦克风）

选一个你日常用的 IM 工具：
- 飞书（推荐，企业场景友好）
- Telegram
- WhatsApp
- Discord

绑定后，你就可以像发消息一样给龙虾下指令。

## 三、部署方案：三种选择，对号入座

### 方案一：桌面一键安装 ⭐ 新手首选

**适合谁**：零代码基础，想快速体验

**优点**：
- 不用配服务器
- 上手门槛低
- 零成本（只付模型 API 费）

**操作步骤**（以 OneClaw 为例）：
1. 去 OneClaw 官网下载对应系统安装包
2. 双击安装，按提示完成
3. 在软件内选择大模型，粘贴 API Key
4. 绑定飞书或 Telegram 账号

全程 10 分钟搞定。我第一个龙虾就是这么起的。

### 方案二：云端托管服务 ⭐ 懒人首选

**适合谁**：不想维护服务器，希望 24 小时在线

**推荐服务**：
| 服务 | 价格 | 特点 |
|------|------|------|
| Kimi Claw | ¥199/月 | 一键部署，Allegretto 及以上套餐 |
| MaxClaw | 订阅包含 | Minimax 托管，网页端集成 |

**优点**：免部署、免运维、开箱即用

**缺点**：灵活性稍差，定制能力有限

如果你只是想用，不想折腾，选这个最省心。

### 方案三：云服务器部署 ⭐ 技术党首选

**适合谁**：开发者、技术爱好者、有深度定制需求

**优点**：
- 完全自主可控
- 可深度定制
- 数据安全有保障

**成本**：服务器约$2/月 + 模型 API 费用

**操作步骤**：
```bash
# 1. 准备服务器（阿里云/Zeabur，推荐 Ubuntu）
# 2. 安装 Docker
curl -fsSL https://get.docker.com | sh

# 3. 运行 OpenClaw
docker run -d --name openclaw -p 8080:8080 openclaw/core

# 4. 配置反向代理（可选，绑定域名）
```

这个方案自由度最高，但需要一点技术基础。

## 四、核心配置：让龙虾真正"认识你"

部署成功只是第一步。真正让龙虾好用的，是配置环节。

这步就像培训新员工：你得告诉它你是谁、做什么工作、有什么习惯。

### 1. 身份录入（永久记忆，关键步骤）

在 OpenClaw 里输入以下指令（替换成你的情况）：

```
我是自媒体创作者，日常需要你帮我整理资料、撰写初稿、管理日程、查询信息。
我习惯简洁直接的回复，每天上午 9 点向我汇报当天待办事项。
以上内容请永久记住。
```

**重点**：
- 说清楚你的职业/角色
- 列出核心需求
- 说明沟通偏好
- 设定定时任务

这一步做好了，后续使用会顺畅很多。

### 2. 安装技能（给龙虾装手脚）

OpenClaw 有个 ClawHub 技能市场，类似手机应用商店。

**新手必装的 4 个核心技能**：

| 技能 | 功能 | 为什么装 |
|------|------|---------|
| **Capability Evolver** | 自我优化复盘 | 越用越聪明，自动记录经验 |
| **Agent Browser** | 自主操作浏览器 | 自动查信息、抓取数据 |
| **Summarize** | 快速总结文档 | 节省阅读时间，支持 PDF/网页 |
| **Find Skills** | 自动搜索技能 | 按需扩展功能，不用手动找 |

**安装方式**：
OpenClaw Web 控制台 → ClawHub 市场 → 搜索技能名 → 一键安装

我靠这几个技能，把日常工作效率提升了至少 50%。

### 3. 对接工作渠道（无缝指挥）

以飞书为例：

1. 飞书开放平台创建"企业自建应用"
2. 开启机器人权限和消息接收权限
3. 获取 App ID 和 App Secret
4. 在 OpenClaw 配置页面填写信息，完成绑定

绑定后，你就可以在飞书里直接@龙虾，它会响应并执行任务。

## 五、实战指令：像指挥同事一样自然

配置完成后，使用非常简单。就像和同事聊天一样，用自然语言下指令。

### 办公自动化
```
把我邮箱里所有来自'客户'的未读邮件，摘要整理成一个表格，发到指定飞书群里。
```

### 信息处理
```
监控 A、B、C 三个竞品的官网和社交媒体，每天下午 5 点，给我发一份它们的动态摘要。
```

### 内容创作
```
根据我昨天整理的会议纪要，生成一份公众号推文大纲，风格要活泼，适合年轻人阅读。
```

### 定时任务
```
每周一早上 9 点，提醒我更新项目周报，顺便帮我整理上周的核心工作内容。
```

**使用技巧**：
- 指令越具体，执行越准确
- 可以设定输出格式（表格、列表、摘要等）
- 支持定时任务和周期性任务
- 复杂任务可以拆解成多步

## 七、避坑指南：这些弯路我替你走过了

### 安全第一
- 设置操作权限白名单
- 推荐用 Docker 沙箱模式
- 敏感数据不要直接给龙虾

### 成本控制
- 设置用量监控，防止 API 费用爆炸
- 新手先用免费额度多的模型
- 或选择固定订阅制服务

### 从简开始
- 别一上来就搞复杂工作流
- 从一个核心需求开始（比如自动整理邮件）
- 跑通后再慢慢扩展
