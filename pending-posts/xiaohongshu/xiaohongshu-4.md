## 💻 技术实现（超简单！）

```javascript
// 核心逻辑就是 prompt 模板
const templates = [
  "{数字}{结果}｜{主题}实战分享{emoji}",
  "从 0 到{数字}｜{时间}实现{结果}的{主题}📈",
  "{主题}实战：我用{方法}实现{结果}{emoji}",
  "普通人如何{结果}？我的{主题}月收入{数字}实录💸",
  "别再说{痛点}了！我的{结果}：{数字}{emoji}"
];

function generateHeadlines(topic, data) {
  return templates.map(t => 
    t.replace('{主题}', topic)
     .replace('{数字}', data.number)
     .replace('{结果}', data.result)
     .replace('{时间}', data.time)
     .replace('{方法}', data.method)
     .replace('{痛点}', data.painPoint)
     .replace('{emoji}', getRandomEmoji())
  );
}
```

**开发时间：** 2 小时
**代码行数：** 约 300 行
**定价：** ¥3.05
