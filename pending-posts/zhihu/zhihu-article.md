## 三、微信接入方案

### 3.1 功能特性

微信自动回复支持：
- 关键词自动回复（精确/模糊匹配）
- AI 智能对话（上下文理解）
- 群聊欢迎语、入群审核
- 定时消息推送
- 消息统计与分析

### 3.2 部署步骤

**第一步：环境准备**
```bash
# 安装 OpenClaw
npm install -g openclaw

# 安装微信自动化技能
openclaw skills install openclaw-china-wechat-auto-reply
```

**第二步：配置微信**
```yaml
# ~/.openclaw/wechat-config.yaml
wechat:
  phone: "13800138000"
  auto_reply: true
  ai_model: "qwen-plus"
  keywords:
    - trigger: "价格"
      response: "我们的产品定价如下..."
```

**第三步：启动服务**
```bash
openclaw wechat-auto-reply start
```

### 4.2 部署步骤

**第一步：创建钉钉应用**
1. 访问 https://open.dingtalk.com
2. 创建企业内部应用
3. 获取 AgentId、AppKey、AppSecret

**第二步：配置技能**
```yaml
# ~/.openclaw/dingtalk-config.yaml
dingtalk:
  agent_id: "123456"
  app_key: "dingxxxxx"
  app_secret: "xxxxx"
  
  chatbot:
    keywords:
      - trigger: "帮助"
        response: "请输入您的问题..."
```

**第三步：启动服务**
```bash
openclaw dingtalk-cs start
```

### 5.2 部署步骤

**第一步：创建飞书应用**
1. 访问 https://open.feishu.cn
2. 创建企业内部应用
3. 获取 App ID、App Secret

**第二步：配置多维表格自动化**
```yaml
# ~/.openclaw/feishu-config.yaml
feishu:
  app_id: "cli_xxxxx"
  app_secret: "xxxxx"
  
  bitable_automation:
    tables:
      - app_token: "bascnxxxxx"
        triggers:
          - type: "record_created"
            action: "notify"
```

**第三步：启动服务**
```bash
openclaw feishu-automation start
```
