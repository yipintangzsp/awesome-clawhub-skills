## 一、使用前准备：备好 3 样"工具"，避免手忙脚乱

开始"养龙虾"前，不用准备复杂设备，先把这 3 样东西备齐，后续操作会顺畅很多：

### 1. 运行环境
一台普通电脑就够（Windows 10 及以上、macOS 10.15 及以上、Linux 均可），如果想让龙虾 7×24 小时在线，也可以用云服务器。

### 2. AI 模型（"龙虾的大脑"）
需要一个大模型的 API Key 来驱动 OpenClaw，新手和有经验的用户可以按需选择：

| 类型 | 推荐 | 特点 |
|------|------|------|
| 国内模型（新手首选） | MiniMax | 免费额度多，性价比高 |
| 国内模型 | Kimi | 访问速度快，操作便捷 |
| 国内模型 | 智谱 AI | 稳定耐用 |
| 国际模型 | Claude/GPT-4 | 能力更强，适合进阶用户 |

### 3. 通讯渠道（"指挥龙虾的麦克风"）
选一个你日常常用的即时通讯工具，用来给龙虾发指令：
- 飞书
- Telegram
- WhatsApp
- Discord

## 二、选择部署方案：3 种方式，按需挑选最省心

备齐工具后，就可以开始部署 OpenClaw 了。这里有 3 种方案，覆盖新手、懒人、技术党，大家根据自己的情况选，不用盲目追求复杂。

### 方案一：桌面一键安装 ⭐ 最推荐新手

**适合人群：** 零代码基础、想快速体验的个人用户

**核心优点：**
- 无需配置服务器和复杂环境
- 上手门槛极低
- 零成本（只需承担模型 API 费用）

**具体操作（以 OneClaw 为例）：**
1. 下载客户端：打开 OneClaw 官网，根据系统下载对应安装包
2. 安装并运行：双击安装包，按提示完成安装
3. 配置 API Key：在软件内选择大模型，粘贴 API Key
4. 连接通讯工具：绑定飞书或 Telegram 账号

---

### 方案二：云端托管服务 ⭐ 最省心，懒人首选

**适合人群：** 不想维护服务器、希望 7×24 小时在线的用户

**核心优点：**
- 免部署、免运维
- 开箱即用
- 费用透明可控

**推荐服务：**
| 服务 | 价格 | 特点 |
|------|------|------|
| Kimi Claw | ¥199/月 | 一键部署，Allegretto 及以上套餐 |
| MaxClaw | 包含在订阅 | Minimax 托管，集成在网页端 |

---

### 方案三：云服务器部署 ⭐ 最灵活，适合技术党

**适合人群：** 开发者、技术爱好者、有深度定制需求的企业用户

**核心优点：**
- 自主可控
- 可深度定制
- 数据安全有保障

**成本：** 约$2/月起（服务器费用）

**具体操作：**
1. 准备服务器：阿里云/Zeabur 等平台，推荐 2 核 4G，Ubuntu 系统
2. 安装 Docker：执行对应命令安装 Docker 环境
3. 运行 OpenClaw：输入 Docker 命令启动服务
4. 配置反向代理（可选）：绑定域名，方便访问

## 三、核心配置：让龙虾"认识你"，学会替你干活

部署成功只是第一步，真正让龙虾好用的，是后续的配置。这一步就像"调教"员工，让它记住你的习惯、掌握必备技能，才能真正帮你省时间。

### 1. 身份录入（永久记忆，核心步骤）

这是"养龙虾"的关键，相当于给龙虾设定"工作准则"。

**示例指令：**
```
我是 [你的职业，比如自媒体创作者/职场白领]，日常需要你帮我 [整理资料、撰写初稿、管理日程、查询信息]。我习惯简洁直接的回复，每天上午 9 点向我汇报当天待办事项。以上内容请永久记住。
```

### 2. 安装技能（"给龙虾装手脚"）

新手先安装这 4 个核心技能，足够满足日常需求：

| 技能 | 功能 | 推荐理由 |
|------|------|---------|
| Capability Evolver | 自我优化复盘 | 越用越聪明 |
| Agent Browser | 自主操作浏览器 | 自动查询信息 |
| Summarize | 快速总结文档 | 节省阅读时间 |
| Find Skills | 自动搜索技能 | 按需扩展功能 |

**安装方式：** OpenClaw Web 控制台 → ClawHub 市场 → 搜索技能名称 → 一键安装

### 3. 对接工作渠道（无缝指挥）

以飞书为例：
1. 飞书开放平台创建"企业自建应用"
2. 开启机器人权限和接收消息权限
3. 获取 App ID 和 App Secret
4. OpenClaw 配置页面填写对应信息，完成绑定

## 四、开始使用：像指挥同事一样，下达自然指令

配置完成后，像和同事聊天一样，用自然语言给龙虾下达指令就行。

**常用指令示例：**

### 办公自动化
```
把我邮箱里所有来自'客户'的未读邮件，摘要整理成一个表格，发到指定飞书群里。
```

### 信息处理
```
监控 A、B、C 三个竞品的官网和社交媒体，每天下午 5 点，给我发一份它们的动态摘要。
```

### 内容创作
```
根据我昨天整理的会议纪要，生成一份公众号推文大纲，风格要活泼，适合年轻人阅读。
```

### 定时任务
```
每周一早上 9 点，提醒我更新项目周报，顺便帮我整理上周的核心工作内容。
```
