## 🎯 如何创建第一个 Skill？

### 步骤 1：找需求

**方法：**
- 看 ClawHub 热门榜单
- 看知乎/小红书热门问题
- 看自己的痛点

**案例：**
- 问题："如何识别土狗币？" → Skill：新币保命扫描器
- 问题："小红书标题怎么写？" → Skill：爆款标题魔法师
- 问题："如何选亚马逊产品？" → Skill：亚马逊选品助手

---

### 步骤 2：写脚本

**要求：**
- Python/Node.js 脚本
- 实现核心功能
- 有输入输出

**案例（简化版）：**
```python
# 新币保命扫描器
def scan_token(contract_address):
    # 检查 LP 锁定
    lp_locked = check_lp_lock(contract_address)
    
    # 检查合约权限
    permissions = check_permissions(contract_address)
    
    # 检查持币分布
    distribution = check_distribution(contract_address)
    
    # 计算风险评分
    risk_score = calculate_risk(lp_locked, permissions, distribution)
    
    return {
        "risk_score": risk_score,
        "recommendation": "买入" if risk_score < 30 else "观望"
    }
```

---

### 步骤 3：包装成 Skill

**文件结构：**
```
my-skill/
├── SKILL.md          # Skill 描述
├── main.py           # 主脚本
├── requirements.txt  # 依赖
└── README.md         # 使用说明
```

**SKILL.md 模板：**
```markdown
---
name: my-skill
description: 一句话描述功能
---

# Skill 名称

## 使用方式

```bash

```

## 收费

¥X/次

## 输出内容

| 项目 | 说明 |
|------|------|
| ... | ... |
```
