## 📝 好 PS vs 烂 PS

### 烂 PS
```
我负责了一个项目，帮助公司提高了销售额。
在这个过程中，我学到了很多。
我希望贵校能帮助我实现梦想。
```

**问题：**
- ❌ 动词弱（"负责"、"帮助"）
- ❌ 无数值
- ❌ 无具体行动
- ❌ 模板化

**结果：** 招生官看了想睡觉

---

### 好 PS
```
Spearheaded a cross-functional sales optimization project, 
analyzing customer data to identify 3 key growth opportunities. 
Implemented targeted marketing strategies that increased quarterly 
revenue by 35% ($2.3M) and expanded market share by 8 percentage 
points within 6 months.
```

**优点：**
- ✅ 强动词（Spearheaded）
- ✅ 量化成果（35%、$2.3M、8 个百分点）
- ✅ 具体行动（数据分析、营销策略）
- ✅ 时间框架（6 个月内）

**结果：** 招生官眼前一亮

## 🎯 STAR 法则框架

### S - Situation（情境）

**描述背景和挑战。**

```
❌ "我在一家公司实习"
✅ "在一家 A 轮创业公司，面临用户增长停滞的挑战"
```

**要点：**
- 说明公司/组织背景
- 描述面临的挑战
- 让读者理解情境

---

### T - Task（任务）

**说明你的职责和目标。**

```
❌ "我的任务是提高销售额"
✅ "我的任务是在 3 个月内将季度销售额提升 20%"
```

**要点：**
- 明确你的角色
- 说明具体目标
- 最好有量化指标

---

### A - Action（行动）

**详述你采取的具体行动。**

```
❌ "我做了一些市场调研"
✅ "我分析了 5,000+ 用户数据，识别出 3 个关键增长机会，
   并设计了针对性的营销策略"
```

**要点：**
- 用强动词开头
- 说明具体做了什么
- 展示你的思考和决策

---

### R - Result（结果）

**量化成果和影响。**

```
❌ "销售额提高了"
✅ "季度销售额提升 35%（$2.3M），市场份额扩大 8 个百分点，
   获得 CEO 特别奖"
```

**要点：**
- 用数据说话
- 说明影响范围
- 提及获得的认可

## 📊 修改案例对比

### 案例 1：实习经历

**优化前：**
```
我在一家互联网公司实习，负责数据分析工作。
我帮助团队提高了效率，学到了很多。
```

**优化后：**
```
Developed an automated data analysis pipeline using Python, 
reducing manual processing time by 75% (from 20 hours to 5 hours 
per week). Presented findings to senior leadership, leading to 
3 process improvements that saved $50K annually.
```

**改进点：**
- ✅ 强动词：Developed
- ✅ 量化：75%、20 小时→5 小时、$50K
- ✅ 具体行动：Python 自动化管道
- ✅ 影响：3 个流程改进

---

### 案例 2：科研经历

**优化前：**
```
我参与了一个科研项目，研究人工智能。
我负责写代码，最后发表了论文。
```

**优化后：**
```
Designed and implemented a novel deep learning architecture 
for image classification, achieving 95% accuracy (5% improvement 
over baseline). Co-authored a paper published at CVPR 2025 
(top-tier conference, acceptance rate < 25%).
```

**改进点：**
- ✅ 强动词：Designed, Co-authored
- ✅ 量化：95% 准确率、5% 提升
- ✅ 具体：深度学习架构
- ✅ 成果：CVPR 2025 论文

---

### 案例 3：社团活动

**优化前：**
```
我是学生会主席，组织了很多活动。
这些活动很成功，我锻炼了领导能力。
```

**优化后：**
```
Led a team of 15 students to organize 10+ campus events, 
attracting 2,000+ participants total. Secured $5,000 in 
sponsorships from local businesses and increased student 
engagement by 40% compared to previous year.
```

**改进点：**
- ✅ 强动词：Led
- ✅ 量化：15 人团队、10+ 活动、2,000+ 参与者、$5,000、40%
- ✅ 具体：拉赞助、提升参与度
- ✅ 对比：与去年对比

## 📋 PS 检查清单

写完 PS 后，问自己：

- [ ] 用了强动词吗？
- [ ] 量化成果了吗？
- [ ] 有具体行动吗？
- [ ] 有时间框架吗？

- [ ] 无语法错误吗？
- [ ] 字数符合要求吗？

**8 个全部 OK 再提交。**

## 💡 常见错误

### 错误 1：泛泛而谈
```
❌ "我学到了很多"
✅ "我掌握了 Python 数据分析和机器学习技能"
```

### 错误 2：罗列经历
```
❌ "我做了 A，做了 B，做了 C"
✅ "通过 A 项目，我发现了 B 问题，用 C 方法解决"
```

### 错误 3：过度吹嘘
```
❌ "我是最优秀的人才"
✅ "这段经历让我成长为..."
```

### 错误 4：模板化
```
❌ "从小我就梦想..."
✅ 用具体故事开头
```

## 💡 最后建议

1. **早开始** - PS 需要反复修改，至少预留 2 个月
2. **多找人看** - 教授、学长、朋友，越多反馈越好
3. **持续修改** - 好 PS 是改出来的，不是写出来的
4. **保持真实** - 不要编造经历，招生官能看出来
