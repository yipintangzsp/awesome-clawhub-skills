## 🤖 为什么 AI 总说废话？

你是不是也这样：

```
你："帮我写个文案"
AI："好的，请问您需要什么类型的文案？..."

你："分析一下这个市场"
AI："当然，为了给您更准确的分析..."

你："做个策划方案"
AI："我很乐意帮助您..."
```

**AI 说了很多，但什么都没说。**

问题不在 AI，在你的 Prompt 太烂。

## 🎯 好 Prompt vs 烂 Prompt

### 烂 Prompt
```
"帮我写个文案"
```

**问题：**
- ❌ 没有人设
- ❌ 没有约束
- ❌ 没有格式
- ❌ 质量不可控

---

### 好 Prompt
```
# Role
你是一名资深文案专家，拥有 10 年广告行业经验

# Task
为一款减肥产品写小红书文案

# Constraints
- 标题必须有情绪词和 Emoji
- 正文必须包含 3 个用户痛点
- 必须有行动号召
- 字数 300-500 字

# Output Format
1. 标题（5 个选项）
2. 正文
3. 热门标签（5 个）

# Quality Standards
- 专业准确
- 逻辑清晰
- 实用可执行
```

**效果：** AI 输出质量提升 10 倍

## 📝 Prompt 优化框架

### 1️⃣ Role - 人设设定

**给 AI 一个身份。**

```
❌ 没有人设
✅ "你是一名资深文案专家，拥有 10 年广告行业经验"
✅ "你是一名麦肯锡高级合伙人"
✅ "你是一名哈佛招生官"
```

**为什么有效？**
- AI 会模仿该角色的思维方式
- 输出更专业
- 语气更准确

---

### 2️⃣ Task - 任务描述

**明确告诉 AI 要做什么。**

```
❌ "帮我写个文案"
✅ "为一款减肥产品写小红书文案，目标用户是 25-35 岁女性"
```

**为什么有效？**
- 减少歧义
- AI 知道具体要什么
- 输出更精准

---

### 3️⃣ Constraints - 约束条件

**给 AI 设定边界。**

```
- 必须分步骤思考，展示推理过程
- 输出必须结构化，使用标题和列表
- 必须给出具体可执行的建议
- 如有不确定，明确说明假设条件
```

**为什么有效？**
- 防止 AI 瞎扯
- 输出更规范
- 质量可控

---

### 4️⃣ Output Format - 输出格式

**告诉 AI 怎么输出。**

```
输出格式：
1. 【问题分析】- 理解用户需求
2. 【核心思路】- 解决框架
3. 【详细方案】- 分步骤执行
4. 【注意事项】- 风险提示
5. 【延伸建议】- 额外价值
```

**为什么有效？**
- 输出结构化
- 易于阅读
- 方便使用

---

### 5️⃣ Quality Standards - 质量标准

**告诉 AI 什么是好。**

```
- 专业准确，不胡说
- 逻辑清晰，不啰嗦
- 实用可执行，不空泛
- 如有数据，注明来源或估算依据
```

**为什么有效？**
- AI 知道什么是好
- 自我检查
- 质量更高
