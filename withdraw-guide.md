## 🏦 提现步骤

### 步骤 1：登录 ClawHub 后台
```
1. 打开 https://clawhub.ai
2. 登录创作者账号
3. 进入「创作者中心」
4. 点击「收入结算」
```

### 步骤 2：查看可提现金额
```
- 查看「可提现余额」
- 确认最低提现门槛（通常¥100-500）
- 查看结算周期（T+7 或月结）
- 查看手续费（通常 3-10%）
```

### 步骤 3：绑定收款账户
```
支持方式：
□ 支付宝
□ 微信
□ 银行卡
□ 加密货币钱包（BNB Chain BEP20）

建议：加密货币到账最快（几小时）
```

### 步骤 4：申请提现
```
1. 点击「申请提现」
2. 输入提现金额（需≥门槛）
3. 选择收款方式
4. 确认提交
```

### 步骤 5：等待打款
```
- 法币（支付宝/微信/银行卡）：1-7 工作日
- 加密货币（BNB Chain）：几小时
```
