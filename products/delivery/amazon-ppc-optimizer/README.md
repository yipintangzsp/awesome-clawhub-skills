## 分类
电商/广告优化

## 价格
¥99/月

## 安装

```bash
clawhub install amazon-ppc-optimizer
```

## 使用

```bash
amazon-ppc-optimizer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/amazon-ppc-optimizer.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 99,
  "amazon_ads_api_key": "your_amazon_api_key",
  "marketplace": "US",
  "target_acos": 25
}
```
