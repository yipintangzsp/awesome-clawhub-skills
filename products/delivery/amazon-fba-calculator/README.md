# amazon-fba-calculator

亚马逊 FBA 计算器（¥99/月）

## 分类
电商/亚马逊

## 价格
¥99/月

## 安装

```bash
clawhub install amazon-fba-calculator
```

## 使用

```bash
amazon-fba-calculator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/amazon-fba-calculator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9,
  "marketplace": "US",
  "category": "Electronics"
}
```

## 功能

- FBA 费用计算
- 利润率分析
- 竞品对比
- 定价建议
- SkillPay 集成
