# Amazon Keyword Tracker - 亚马逊关键词追踪

📊 掌握排名，优化流量

## 核心功能

| 功能 | 说明 | 价值 |
|------|------|------|
| 排名追踪 | 实时监控关键词排名 | 了解 SEO 效果 |
| 历史趋势 | 排名变化曲线 | 发现规律 |
| 竞品对比 | 多 ASIN 对比 | 找到差距 |
| 自动告警 | 排名波动提醒 | 及时应对 |
| 报告导出 | CSV/PDF 报告 | 团队汇报 |

## 关键词类型

```
核心词 (1-2 字): 耳机
长尾词 (3-5 字): 无线蓝牙耳机 降噪
品牌词：Sony 耳机
竞品词：AirPods 平替
```

## 示例

```bash
# 追踪关键词
/amazon-keyword-tracker --asin "B08XXX" --keywords "wireless earbuds,bluetooth headphones" --marketplace US

# 竞品对比
/amazon-keyword-tracker --action compare --asins "B08XXX,B09YYY" --keywords "noise cancelling earbuds"

# 批量追踪
/amazon-keyword-tracker --action bulk --file keywords.csv --asin "B08XXX"
```
