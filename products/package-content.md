### 01-环境搭建教程.md

```markdown
# 环境搭建教程

## 前置要求

- macOS / Windows / Linux
- Node.js v18+
- 基础命令行知识

## 步骤 1：安装 Node.js

下载地址：https://nodejs.org/

安装后验证：
```bash
node -v
npm -v
```

## 步骤 2：安装 OpenClaw

```bash
npm install -g openclaw
```

## 步骤 3：配置工作区

```bash
cd ~
mkdir -p .openclaw/workspace
cd .openclaw/workspace
```

## 步骤 4：运行第一个 Skill

```bash
cd 02-24 个 Skill 源码/链上工具类/airdrop-checker
npm install
node index.js 0x你的钱包地址
```

## 常见问题

**Q：npm install 报错**
A：检查 Node.js 版本，需要 v18+

**Q：命令找不到**
A：确保在正确的目录

**Q：运行没反应**
A：检查配置文件是否创建

---

有问题微信群问～
```

## 打包步骤

### 方法 1：本地打包

1. 创建文件夹结构
2. 复制所有 Skill 源码
3. 添加教程文件
4. 压缩成 ZIP
5. 上传百度网盘

### 方法 2：GitHub 私有仓库

1. 创建 GitHub 私有仓库
2. 推送所有内容
3. 付费后邀请用户

### 方法 3：蓝奏云（推荐）

1. 注册蓝奏云（免登录下载）
2. 上传 ZIP 文件
3. 获取分享链接

## 网盘选择对比

| 平台 | 优点 | 缺点 |
|------|------|------|
| 百度网盘 | 用户多 | 下载限速 |
| 蓝奏云 | 免登录/不限速 | 免费 100MB |
| 天翼云盘 | 速度快 | 用户少 |
| GitHub | 开发者友好 | 需要梯子 |
| 阿里云盘 | 速度快 | 分享限制 |

**推荐**：蓝奏云（小文件）+ 百度网盘（大文件）
