# 🪂 Airdrop Checker - 空投资格检测器

**一键检查钱包空投资格，不错过任何空投机会**

---

## 😫 痛点

- 链上项目太多，记不住哪些需要交互
- 空投资格标准复杂，手动检查耗时费力
- 等发现时快照已结束，错失几千刀空投
- 不知道如何优化钱包以满足空投要求

---

## ✨ 功能

- **一键检测** - 输入钱包地址，自动扫描 20+ 热门空投项目
- **资格评估** - 显示每个项目的当前资格状态（符合/边缘/不符合）
- **价值预估** - 基于历史数据和社区预期，预估空投价值
- **优化建议** - 针对性建议如何操作以满足空投要求
- **定期提醒** - 可设置定期检查，跟踪资格变化

## 🎯 使用场景

### 场景 1：新钱包空投规划
刚创建新钱包，想知道应该交互哪些项目来最大化空投资格。

### 场景 2：定期检查资格
每周检查一次，确保没有错过新的空投机会或资格变化。

### 场景 3：快照前紧急检查
听说某项目要快照，快速检查自己是否符合资格。

### 场景 4：多钱包管理
管理多个钱包，批量检查每个钱包的空投资格状态。

### 场景 5：优化建议执行
根据优化建议完成特定操作，提升空投资格。
