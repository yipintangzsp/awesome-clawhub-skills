# 介绍

**nexu**（next to you）是将 **OpenClaw 🦞** Agent 接入飞书、Slack、Discord 等 IM 的开源桌面客户端（Electron）。

提供图形化配置、内置飞书 Skills、多模型支持与 BYOK。

配置与运行状态主要保存在本机。
