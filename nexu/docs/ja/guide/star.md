# GitHub で Star する

nexu が役に立ったなら、GitHub で ⭐ をお願いします。チームの励みになり、より多くの人にプロジェクトを知ってもらえます。

## なぜ Star するの？

- **認知度** — Star はより多くの開発者やチームが nexu を見つける助けになります
- **モチベーション** — すべての Star がチームの開発スピードを後押しします
- **最新情報** — Star したリポジトリは後から簡単に見つけられます
