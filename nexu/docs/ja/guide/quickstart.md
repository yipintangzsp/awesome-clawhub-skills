# 1分クイックスタート

## ステップ 1：ダウンロードとインストール

[nexu クライアント](https://nexu.io/download)をダウンロードし、インストールしてダブルクリックで起動します。

![nexu ホーム画面](/assets/nexu-home.webp)

## ステップ 2：モードを選択

左サイドバーの**設定**を開き、統合モードを選択します：

- **サインイン** — **Nexu Official** を選択し、**Nexu にサインイン**をクリックすると、高品質モデルがすぐに利用可能
- **BYOK モード** — プロバイダー（Anthropic、OpenAI など）を選択し、API Key を貼り付けて**保存**をクリック

![モデル統合モードの選択](/assets/nexu-settings-open.webp)

![Nexu Official — サインインですぐ利用](/assets/nexu-models-official.webp)

## ステップ 3：チャンネルを設定

チャンネルを1つ選んでセットアップを完了します：

- [Feishu](/ja/guide/channels/feishu) — App ID と App Secret のみ必要
- [Slack](/ja/guide/channels/slack) — マニフェストでワンクリックアプリ作成
- [Discord](/ja/guide/channels/discord) — Application ID と Bot Token が必要

設定が完了したら「チャット」をクリックしてエージェントとの会話を開始 🎉
