# はじめに

**nexu**（next to you）は、**OpenClaw 🦞** エージェントを Feishu、Slack、Discord などの IM チャンネルに接続するオープンソースデスクトップクライアント（Electron）です。

グラフィカルなセットアップ、組み込みの Feishu Skills、マルチモデル対応、BYOK をサポートしています。

設定とランタイム状態は主にローカルマシンに保存されます。
