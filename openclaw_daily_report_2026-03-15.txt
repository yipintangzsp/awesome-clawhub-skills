【今日概览】
OpenClaw 作为本地 AI 自动化框架，持续为用户提供安全、可控的 AI 助手部署方案。

【核心功能模块】

1. 文件与内容管理
   - 本地文件读写编辑 (read/write/edit)
   - 支持文本、图片、PDF 多格式处理
   - Workspace 作为统一工作目录

2. 网络与浏览器自动化
   - web_search / web_fetch 信息检索
   - browser 工具完整浏览器控制
   - 支持伪终端 (PTY) 执行复杂 CLI

3. 消息与通知系统
   - 多通道消息发送 (Feishu/Discord/WhatsApp)
   - cron 定时任务与提醒
   - TTS 语音输出

4. Feishu 深度集成
   - 文档 (docx) 读写与协作
   - 云盘 (Drive) 文件管理
   - 知识库 (Wiki) 导航
   - 多维表格 (Bitable) CRUD 操作
   - 聊天机器人集成

5. 子代理与任务分发
   - sessions_spawn 创建隔离子代理
   - 支持 runtime="subagent" 和 runtime="acp"
   - 任务完成自动通知

6. 设备与节点控制
   - nodes 工具管理配对设备
   - 相机快照、屏幕录制、位置获取
   - 通知推送与管理

【典型使用场景】

✅ 自媒体内容自动化
   - 定时抓取热点 → 生成草稿 → Feishu 文档协作

✅ 跨境电商数据监控
   - 链上数据查询 → 价格提醒 → 邮件/消息通知

✅ 个人知识库管理
   - PDF/图片分析 → 记忆系统归档 → Wiki 结构化存储

✅ 安全运维
   - 定期 healthcheck → 防火墙/SSH 审计 → 风险报告

【开发者友好特性】

• 技能热加载：clawhub CLI 安装/更新/发布
• 配置管理：gateway config.patch 安全更新
• 会话隔离：subagent 独立执行环境
• 审批机制：高风险操作需用户确认

【安全与隐私】

⚠️ 外部操作确认机制：
   - 发送邮件/公开消息 → 需确认
   - 删除文件 → 使用 trash 而非 rm
   - 不确定操作 → 暂停询问
