## 已整理的使用案例

### 📋 热门使用场景

**1. 代码开发工作流**
- PR 自动审查 → 完成代码后自动提交 PR，OpenClaw 审查 diff 并在 Telegram 回复修改建议
- GitHub + Telegram 联动 - 提交、审查、合并通知一站式

**2. 个人技能定制**
- Wine Cellar Skill - 葡萄酒收藏管理（几分钟内创建的技能）
- 各种自定义技能：待办事项、习惯追踪、数据查询等

**3. 多平台消息聚合**
- WhatsApp / Telegram / Discord / iMessage / Slack 统一接入
- 从一个入口管理所有消息渠道

**4. 自动化任务**
- Cron 定时任务 - 每天早上发送天气 + 日程提醒
- Heartbeat 心跳检查 - 定期检查邮箱、日历、项目状态
- Gmail 推送通知 - 新邮件实时提醒

**5. 语音与 Canvas**
- 语音通话（macOS/iOS/Android）
- Canvas 实时渲染控制界面

**6. 家庭/个人助理**
- 日程管理、提醒、待办
- 家庭设备控制（通过节点）
- 摄像头监控、屏幕录制
