# 🖼️ NFT Floor Monitor - NFT 地板价监控

**核心功能：**
- 多平台比价 - OpenSea/Blur/Looksrare 实时地板价
- 走势分析 - 7 天/30 天价格趋势（最高/最低/平均）
- 成交量监控 - 24h 成交量、买卖比、活跃度
- 稀有度分析 - 地板价对应稀有度排名，判断性价比
- 交易信号 - 智能买入/观望/卖出建议
- 价格提醒 - 设置目标价，跌破自动通知

**定价：** ¥5/次 | ¥49/月 | ¥499/年

**典型场景：**
1. 低价买入机会 - 设置提醒，跌破目标价及时通知
2. 多系列对比 - 同时监控几个系列选最优
3. 稀有度捡漏 - 发现地板价中的高稀有度 NFT
4. 趋势判断 - 分析走势决定买入或观望
5. 卖出决策 - 监控地板价变化决定何时获利了结

**输入示例：** `/nft-floor-monitor BAYC`

**输出示例：**
```
🖼️ BAYC 监控报告
💰 地板价：28.5 ETH（Blur 28.3 最低）
📊 24h：45 ETH 成交 | -2.3% | 买卖比 0.8
📈 7 天：最高 31.2 | 最低 27.8 | 平均 29.5 | ↘️ -3.4%
🎯 稀有度：地板价#7,234/10,000（后 28%）
🔔 信号：🟡 观望（等 27.5-28.0 买入）
💡 建议：理想买入 27.5 ETH，止损 26 ETH，目标 32 ETH
```

**地板价低于 7 天平均可买入，稀有度高的优先！**
