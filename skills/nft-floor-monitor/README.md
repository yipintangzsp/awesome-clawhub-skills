## 💡 使用建议

### ✅ 买入信号
- 地板价低于 7 天平均价
- 成交量放大（有人抄底）
- 趋势开始反转向上
- 稀有度高的在地板价

### ❌ 避免买入
- 地板价持续下跌中
- 成交量极低（无人问津）
- 项目方有负面消息
- 整体市场熊市

### 🎯 稀有度策略
- 地板价通常是稀有度最低的
- 前 30% 稀有度合理溢价 20-50%
- 前 10% 稀有度合理溢价 50-100%
- 前 1% 稀有度可能溢价 200%+
