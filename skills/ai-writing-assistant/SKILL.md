# AI Writing Assistant - AI 写作助手

## Description
专业 AI 写作助手，帮助用户撰写各类文章、报告、邮件等内容。支持多种文体和风格定制，提升写作效率和质量。

## Usage
```bash
# 基础用法
/ai-writing-assistant --type "blog" --topic "AI 工具评测" --length 1000

# 指定风格
/ai-writing-assistant --type "email" --tone "professional" --recipient "client"

# 润色已有内容
/ai-writing-assistant --action "polish" --input "your-text.txt"
```

### 参数说明
- `--type`: 文章类型 (blog, email, report, essay, social)
- `--topic`: 主题/标题
- `--length`: 目标字数
- `--tone`: 语气风格 (professional, casual, friendly, formal)
- `--action`: 操作类型 (write, polish, expand, summarize)
