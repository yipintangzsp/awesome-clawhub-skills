# TikTok Hashtag Generator - TikTok 标签生成

## Description
TikTok 热门标签生成工具，基于视频内容和 trending 话题生成最佳标签组合。提升视频曝光量和进入 For You 页面的几率。

## Usage
```bash
# 生成标签
/tiktok-hashtag-generator --topic "dance challenge" --niche "entertainment"

# 追踪趋势
/tiktok-hashtag-generator --action "trending" --region "US"

# 竞品分析
/tiktok-hashtag-generator --action "analyze" --username "@creator"
```

### 参数说明
- `--topic`: 视频主题
- `--niche`: 内容领域 (dance, comedy, education, beauty, food, fitness, etc.)
- `--region`: 目标地区 (US, UK, CN, global)
- `--action`: 操作类型 (generate, trending, analyze, optimize)
- `--username`: TikTok 用户名
