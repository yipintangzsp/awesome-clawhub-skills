## 使用方式

```bash
# 基础用法
ai-cost-optimizer --usage <使用数据> --providers <云服务商>

# 高级用法
ai-cost-optimizer --providers aws,azure --period 30d --recommendations --output excel
```

## 功能特点

- ⚡ **成本深度分析** - 多维度拆解云费用，识别浪费和异常支出
- 🔄 **多供应商比价** - AWS/Azure/阿里云/腾讯云自动比价，推荐最优方案
- 📊 **使用效率评估** - CPU/内存/存储利用率分析，发现闲置资源
- 🎯 **自动资源调度** - 根据负载自动扩缩容，避免过度配置
- 💡 **优化建议执行** - 预留实例/竞价实例/存储分层，自动执行省钱策略
