# playlist-curator-ai

歌单策划 AI（¥3/次）

## 分类
娱乐休闲

## 价格
¥3/次

## 安装

```bash
clawhub install playlist-curator-ai
```

## 使用

```bash
playlist-curator-ai [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/playlist-curator-ai.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 3
}
```

## 功能

- 自动收费验证
- SkillPay 集成

- 详细日志
