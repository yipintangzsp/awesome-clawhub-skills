# memecoin-launch-monitor

土狗币发射监控（SkillPay 收费版）

## 分类
Web3/加密货币

## 价格
¥39/月

## 安装

```bash
clawhub install memecoin-launch-monitor
```

## 使用

```bash
memecoin-launch-monitor [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/memecoin-launch-monitor.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
