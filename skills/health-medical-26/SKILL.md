# 睡眠监测与改善系统

## 技能描述
睡眠质量管理工具，整合睡眠设备数据、分析睡眠阶段、识别睡眠问题、提供改善方案、白噪音和助眠内容。

## 定价
- **标准版**: ¥299/月
- **专业版**: ¥599/月（+ 深度分析）
- **尊享版**: ¥999/月（+ 睡眠教练）

## 核心功能
1. 睡眠数据自动同步
2. 睡眠阶段分析
3. 睡眠问题识别
4. 个性化改善方案
5. 助眠内容库
6. 睡眠报告

## 使用方式
```bash
/sleep-coach activate
/sleep-coach sync --device oura-ring
/sleep-coach analyze --period 30d
/sleep-coach improve --issue insomnia
```

## 适用场景
- 失眠人群
- 睡眠质量差
- 作息不规律
- 打鼾/呼吸暂停
