# nft-sniper-bot

NFT 狙击机器人（SkillPay 收费版）

## 分类
Web3/加密货币

## 价格
¥99/月

## 安装

```bash
clawhub install nft-sniper-bot
```

## 使用

```bash
nft-sniper-bot [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/nft-sniper-bot.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
