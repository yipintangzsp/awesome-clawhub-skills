# nft-sniper-bot

NFT 狙击机器人（SkillPay 收费版）

## 使用方式

```bash
nft-sniper-bot --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/nft-sniper-bot.json` 配置 SkillPay API Key。
