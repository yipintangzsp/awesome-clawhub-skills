# web3-portfolio-tracker - Web3 多链资产管理追踪器

**痛点**：资产分散在多条链，手动统计耗时，收益计算复杂，无法全局查看
**价值**：一键聚合所有链上资产，实时盈亏统计，投资收益一目了然

## 使用方式

```bash
web3-portfolio-tracker --help
```

## 功能特点

- ⚡ **多链资产聚合** - 支持 EVM/Solana 等 20+ 链，一键查看所有资产
- 📊 **收益自动计算** - 自动追踪 LP 收益、质押奖励、空投收入
- 💰 **盈亏实时统计** - 基于成本价计算持仓盈亏，支持多种计价方式
- 🔒 **历史交易分析** - 分析交易习惯，识别亏损模式，优化投资策略
- 🎯 **资产配置建议** - 基于风险偏好给出资产配置优化建议

## 适用场景

- 多链资产统一管理
- DeFi 投资收益追踪
- 加密资产税务计算
- 投资组合绩效分析

## 配置

在 `~/.openclaw/workspace/config/web3-portfolio-tracker.json` 配置 SkillPay API Key。
