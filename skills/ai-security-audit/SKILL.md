## 使用方式

```bash
# 基础用法
ai-security-audit --system <系统名称> --scope <审计范围>

# 高级用法
ai-security-audit --system ml-pipeline --scope full --compliance GDPR --output pdf
```

## 功能特点

- ⚡ **全面漏洞扫描** - 代码/依赖/配置漏洞自动检测，覆盖 OWASP Top 10
- 🛡️ **模型安全评估** - 对抗攻击/模型投毒/成员推断等 AI 特有风险检测
- 🔐 **数据隐私检查** - 敏感数据识别、加密检查、访问控制审计
- 📋 **合规性审计** - GDPR/网络安全法/数据安全法合规自动评估
- 📊 **风险评级报告** - 风险分级排序，给出修复优先级和建议
