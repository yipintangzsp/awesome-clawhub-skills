# tip-calculator-pro

小费计算器（¥1/次）

## 分类
生活便利

## 价格
¥1/次

## 安装

```bash
clawhub install tip-calculator-pro
```

## 使用

```bash
tip-calculator-pro [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/tip-calculator-pro.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 1
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
