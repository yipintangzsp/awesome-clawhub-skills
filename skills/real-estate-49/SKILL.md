# 房地产政策解读与市场监控系统

## 技能描述
房地产政策追踪工具，全国政策实时监控、政策影响分析、市场数据监控、趋势预测、投资决策支持。

## 定价
- **标准版**: ¥499/月
- **专业版**: ¥999/月
- **机构版**: ¥1999/月

## 核心功能
1. 政策实时追踪
2. 政策影响分析
3. 市场数据监控
4. 趋势预测
5. 城市对比
6. 投资时机建议

## 使用方式
```bash
/property-policy activate
/property-policy monitor --cities Beijing,Shanghai,Shenzhen
/property-policy analyze --policy POLICY123
/property-policy market-report --city Guangzhou --period monthly
```
