# webinar-script-writer

网络研讨会脚本（SkillPay 收费版）

## 分类
内容创作

## 价格
¥49/次

## 安装

```bash
clawhub install webinar-script-writer
```

## 使用

```bash
webinar-script-writer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/webinar-script-writer.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
