# webinar-script-writer

网络研讨会脚本（SkillPay 收费版）

## 使用方式

```bash
webinar-script-writer --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/webinar-script-writer.json` 配置 SkillPay API Key。
