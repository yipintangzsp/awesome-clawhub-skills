# AI Email Writer - AI 邮件代写

## Description
专业商务邮件代写工具，适用于各种商务场景。自动生成得体、专业的邮件内容，提升沟通效率和回复率。

## Usage
```bash
# 写商务邮件
/ai-email-writer --type "follow-up" --recipient "client" --topic "项目进度"

# 写求职邮件
/ai-email-writer --type "cover-letter" --company "目标公司" --position "职位"

# 写销售邮件
/ai-email-writer --type "cold-outreach" --product "产品名称" --tone "friendly"
```

### 参数说明
- `--type`: 邮件类型 (follow-up, inquiry, complaint, apology, cover-letter, cold-outreach, thank-you, resignation)
- `--recipient`: 收件人类型 (client, boss, colleague, hr, customer, partner)
- `--topic`: 邮件主题
- `--tone`: 语气 (professional, friendly, formal, urgent)
- `--language`: 语言 (zh, en, auto)
