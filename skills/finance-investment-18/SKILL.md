# 保险配置智能规划与理赔辅助系统

## 技能描述
AI 保险规划工具，根据家庭情况、财务状况、风险偏好，智能推荐保险配置方案，保单管理、续保提醒、理赔辅助，让保险更透明、更划算。

## 定价
- **标准版**: ¥399/月
- **家庭版**: ¥799/月
- **尊享版**: ¥1499/月（+ 理赔顾问）

## 核心功能
1. 保险需求智能分析
2. 产品对比和推荐
3. 现有保单诊断
4. 续保和缴费提醒
5. 理赔流程辅助
6. 保费优化建议

## 使用方式
```bash
/insurance-planner activate
/insurance-planner analyze --family 4 --income 800000
/insurance-planner compare --type critical-illness
/insurance-planner claims --policy POL123456
```

## 适用场景
- 家庭保险规划
- 企业员工福利
- 保险经纪人工具
- 保单托管服务
