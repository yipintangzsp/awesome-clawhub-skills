# Metaverse Office - 元宇宙办公室

## 概述

为企业提供虚拟办公空间解决方案，打造沉浸式远程协作体验。

## 服务内容包括

- 虚拟办公室设计
- 虚拟化身系统
- 视频会议集成
- 协作工具整合
- 活动空间定制

## 定价

- 月费：¥999/月
- 支持按员工数量定制价格

## 使用方法

```bash
metaverse-office --company <公司名> --employees <员工数>
```

## 配置

在 `~/.openclaw/workspace/config/metaverse-office.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 999
}
```
