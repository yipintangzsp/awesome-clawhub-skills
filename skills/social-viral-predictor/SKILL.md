# 社交媒体爆款预测 - 内容爆款概率分析

## 概述
基于历史爆款数据，预测社交媒体内容的爆款概率。支持多平台：抖音、小红书、微博、公众号。

## 定价
- **按次收费：** ¥5/次
- **包月订阅：** ¥99/月（无限次）

## 功能特性
- 📊 爆款概率预测（0-100%）
- 🔍 内容元素分析
- 🎯 优化建议
- 📈 平台适配建议
- 🔥 热门标签推荐

## 使用方式
```bash
# 基础预测
/viral-predict "内容描述" --platform 抖音

# 多平台对比
/viral-predict "内容" --compare 抖音，小红书，微博

# 优化建议
/viral-predict "内容" --optimize

# 批量预测
/viral-predict "内容列表" --batch
```
