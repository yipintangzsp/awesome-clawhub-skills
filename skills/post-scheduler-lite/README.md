# post-scheduler-lite

帖子调度精简版（¥5/次）

## 分类
社交媒体

## 价格
¥5/次

## 安装

```bash
clawhub install post-scheduler-lite
```

## 使用

```bash
post-scheduler-lite [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/post-scheduler-lite.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 5
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
