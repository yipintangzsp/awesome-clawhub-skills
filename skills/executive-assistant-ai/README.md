# Executive Assistant AI - 高管助理 AI

## 概述

为企业高管提供智能行政助理服务，自动化日常行政工作，提升工作效率。

## 服务内容包括

- 日程安排管理
- 邮件筛选回复
- 会议组织安排
- 文档报告生成
- 差旅安排

## 定价

- 月费：¥999/月
- 支持按工作量定制价格

## 使用方法

```bash
executive-assistant-ai --tasks <任务类型> --priority <优先级>
```

## 配置

在 `~/.openclaw/workspace/config/executive-assistant-ai.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 999
}
```
