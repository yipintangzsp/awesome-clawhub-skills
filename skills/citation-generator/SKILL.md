## 使用方式

```bash
# 基础用法
citation-generator --type journal --style APA --url https://example.com

# 高级用法
citation-generator --batch references.txt --style MLA --output bibtex
```

## 功能特点

- ⚡ **多格式支持** - APA/MLA/Chicago/Harvard/GB/T 等 20+ 格式一键切换
- 📚 **智能识别来源** - 期刊/书籍/网页/视频自动识别，提取元数据
- 🔍 **DOI 自动检索** - 输入 DOI 自动获取完整引用信息
- 📝 **批量生成** - 一次性生成整篇论文参考文献列表
- ✅ **格式校验** - 自动检查格式错误，确保符合期刊要求
