营销/网红营销

## 价格
¥99/月

## 安装

```bash
clawhub install influencer-outreach-bot
```

## 使用

```bash
influencer-outreach-bot [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/influencer-outreach-bot.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 99,
  "platforms": ["instagram", "tiktok", "youtube"],
  "niche": "lifestyle",
  "follower_range": [10000, 100000]
}
```
