# budget-tracker-lite - AI 智能预算追踪系统

**痛点**：钱花哪了不知道，月光存不下钱，预算超支无提醒，理财没规划
**价值**：AI 自动分析消费习惯，制定合理预算，每月多存 30% 收入

## 使用方式

```bash
# 基础用法
budget-tracker-lite --income 10000 --category food --amount 500

# 高级用法
budget-tracker-lite --income 15000 --auto-categorize --monthly-report --saving-goal 5000
```

## 功能特点

- ⚡ **智能支出分类** - AI 自动识别消费类型，餐饮/交通/购物自动归类
- 📊 **消费趋势分析** - 月度/季度消费趋势，找出浪费点
- 💰 **预算智能制定** - 基于收入和习惯，推荐合理预算分配
- 🔔 **超支实时提醒** - 预算快超支时自动提醒，避免月底吃土
- 📈 **储蓄目标追踪** - 设定储蓄目标，可视化追踪进度

## 适用场景

- 💼 **上班族** - 工资理财，月光族变储蓄达人
- 🎓 **学生党** - 生活费管理，培养理财习惯
- 👨‍👩‍👧 **家庭理财** - 家庭收支管理，共同储蓄目标
- 🚀 ** FIRE 计划** - 财务独立提前退休，支出优化
