# crypto-whale-alert-pro

巨鲸警报专业版（SkillPay 收费版）

## 分类
Web3/加密货币

## 价格
¥89/月 | ¥899/年

## 安装

```bash
clawhub install crypto-whale-alert-pro
```

## 使用

```bash
crypto-whale-alert-pro [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/crypto-whale-alert-pro.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 15
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
