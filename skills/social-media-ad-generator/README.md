# social-media-ad-generator

社交媒体广告生成（¥149/月）

## 分类
电商/广告

## 价格
¥149/月

## 安装

```bash
clawhub install social-media-ad-generator
```

## 使用

```bash
social-media-ad-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/social-media-ad-generator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 14,
  "platforms": ["Facebook", "Instagram", "TikTok", "Pinterest"],
  "ad_format": "carousel"
}
```

## 功能

- 多平台广告生成
- 广告文案创作
- 视觉素材建议
- A/B 测试方案
- SkillPay 集成
