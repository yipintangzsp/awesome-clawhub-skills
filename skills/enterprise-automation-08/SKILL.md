# 会议安排与纪要自动生成系统

## 技能描述
智能会议管理系统，自动协调参会人时间、预订会议室、发送邀请、录音转文字、AI 生成会议纪要、行动项追踪，提升会议效率 50%+。

## 定价

- **专业版**: ¥999/月（200 会议）
- **企业版**: ¥1999/月（无限 + 定制集成）

## 核心功能
1. 智能时间协调和冲突检测
2. 会议室自动预订
3. 会议邀请自动发送
4. 录音实时转文字
5. AI 生成会议纪要
6. 行动项自动创建和追踪

## 使用方式
```bash
/meeting-automation activate
/meeting-automation schedule --participants team@company.com
/meeting-automation join --id meeting123
/meeting-automation summary --meeting meeting123
```

- 企业日常会议管理
- 跨部门协作会议
- 客户会议记录
- 远程会议转录
