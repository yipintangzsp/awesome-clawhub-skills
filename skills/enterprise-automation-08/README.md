# 会议安排与纪要自动生成系统

## 产品概述

全流程会议自动化系统，从 scheduling 到 follow-up 一站式管理，AI 自动生成高质量会议纪要，让会议更高效、更有价值。

## 核心功能

### 会议安排
- 智能时间协调
- 会议室自动预订
- 日历同步
- 提醒自动发送

### 会议记录
- 实时录音转文字
- 多语言支持
- 说话人识别
- 关键内容标记

### 纪要生成
- AI 自动总结
- 决策点提取
- 行动项创建
- 自动分发

### 行动追踪
- 任务自动分配
- 截止日期提醒
- 进度追踪
- 下次会议回顾

## 集成
- 飞书日历
- 企业微信
- Outlook
- Zoom/腾讯会议
- 钉钉
