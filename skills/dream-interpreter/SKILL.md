## 使用方式

```bash
# 基础用法
dream-interpreter --dream "梦到被追赶"

# 高级用法
dream-interpreter --dream "梦到考试迟到" --style psychology --detailed
```

## 功能特点

- ⚡ **AI 深度解析** - 结合心理学/精神分析/周公解梦多角度解读
- 🌙 **梦境符号库** - 1000+ 常见梦境符号含义，快速查询
- 🔍 **心理状态分析** - 从梦境推断当前心理状态和压力源
- 📊 **梦境趋势追踪** - 记录梦境历史，发现情绪变化规律
- 💡 **建议与启示** - 基于梦境分析给出生活建议和调整方向

## 适用场景

- 🧠 **自我探索** - 了解潜意识，探索内心世界
- 💆 **压力管理** - 从梦境识别压力源，及时调整
- 📔 **梦境记录** - 养成记录梦境习惯，提升自我认知
- 🔮 **好奇娱乐** - 满足对梦境含义的好奇心
