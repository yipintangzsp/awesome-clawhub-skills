# blog-post-generator

博客文章生成（SkillPay 收费版）

## 分类
内容创作

## 价格
¥29/月

## 安装

```bash
clawhub install blog-post-generator
```

## 使用

```bash
blog-post-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/blog-post-generator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
