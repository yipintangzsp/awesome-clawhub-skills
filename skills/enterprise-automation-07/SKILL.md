# 库存管理与自动补货系统

## 技能描述
智能库存管理系统，实时库存监控、销售预测、自动补货建议、多仓库调拨优化、呆滞库存预警，降低库存成本，提升周转率。

## 定价
- **标准版**: ¥799/月（单仓库）
- **专业版**: ¥1599/月（多仓库）
- **企业版**: ¥2799/月（全渠道 + 供应链）

## 核心功能
1. 实时库存监控和预警
2. AI 销售预测
3. 自动补货计算
4. 多仓库智能调拨
5. 呆滞库存识别
6. 供应商管理和采购自动化

## 使用方式
```bash
/inventory-automation activate
/inventory-automation sync --warehouse all
/inventory-automation forecast --days 30
/inventory-automation reorder --auto true
```

## 适用场景
- 电商库存管理
- 零售连锁库存优化
- 制造业原材料管理
- 跨境电商海外仓
