# ad-copy-ab-test-pro - 广告文案测试专业版

## 技能描述
专业版广告文案 A/B 测试工具，科学设计测试方案，分析测试结果，提供优化建议。支持多变量测试、统计显著性计算、自动化决策。

## 功能特性
- 🧪 A/B 测试方案设计
- 📊 统计显著性计算
- 📈 测试结果分析
- 🤖 自动化决策建议
- 📝 文案变体生成
- 🏆 赢家实施方案

## 使用方式
```bash
# 设计 A/B 测试
/ad-copy-ab-test-pro --design --goal conversion --variants 3

# 计算样本量
/ad-copy-ab-test-pro --sample-size --baseline 2.5 --mde 20

# 分析结果
/ad-copy-ab-test-pro --analyze --variant-a <数据> --variant-b <数据>

# 生成文案变体
/ad-copy-ab-test-pro --generate --product <产品> --count 10
```
