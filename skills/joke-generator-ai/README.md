## 安装

```bash
clawhub install joke-generator-ai
```

## 使用

```bash
joke-generator-ai [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/joke-generator-ai.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 1
}
```
