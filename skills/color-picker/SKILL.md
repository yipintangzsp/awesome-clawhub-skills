## 使用方式

```bash
# 互补色
color-picker "#3498db" complementary

# 类似色
color-picker "#e74c3c" analogous

# 三色搭配
color-picker "#2ecc71" triadic

# 单色渐变
color-picker "#9b59b6" monochromatic
```

## 功能特点

- ⚡ **AI 智能配色** - 基于色彩理论自动生成协调配色方案
- 🎨 **10+ 配色模式** - 互补色/类似色/三色/分裂互补/矩形等
- 🌈 **情绪色彩匹配** - 根据品牌调性（专业/活泼/高端）推荐配色
- 📱 **多场景适配** - UI/平面/室内/服装等多场景配色建议
- 🎯 **可访问性检查** - 自动检查色盲友好性和对比度合规

## 适用场景

- 💻 **UI/UX 设计** - App/网站界面配色，提升用户体验
- 🎨 **平面设计** - 海报/宣传册/Logo 配色设计
- 🏠 **室内设计** - 家居/办公室色彩搭配
- 👗 **时尚设计** - 服装/配饰色彩组合
