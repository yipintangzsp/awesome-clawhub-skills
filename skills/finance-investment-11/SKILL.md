# AI 股票智能分析与交易信号系统

## 技能描述
专业级股票分析系统，实时行情监控、技术指标分析、基本面数据追踪、AI 交易信号生成、风险预警、投资组合优化，适合个人投资者和投资机构。

## 定价
- **标准版**: ¥999/月（A 股/港股）
- **专业版**: ¥1999/月（+ 美股/期货）
- **机构版**: ¥2999/月（+API 接入/定制模型）

## 核心功能
1. 实时行情和技术指标分析
2. 基本面数据自动追踪
3. AI 交易信号生成（买入/卖出/持有）
4. 风险预警和止损建议
5. 投资组合优化
6. 交易回测和策略验证

## 使用方式
```bash
/stock-analysis activate
/stock-analysis track --symbols 00700.HK,AAPL,TSLA
/stock-analysis signal --stock 00700.HK
/stock-analysis portfolio --optimize true
```

## 适用场景
- 个人股票投资
- 量化交易策略
- 投资顾问服务
- 基金选股辅助
