# AI 合同审查与风险识别系统

## 技能描述
专业合同审查工具，AI 自动识别合同风险条款、提示法律问题、提供修改建议、对比标准模板，降低合同法律风险。

## 定价

- **专业版**: ¥1999/月（200 份）
- **企业版**: ¥2999/月（无限 + 定制）

## 核心功能
1. 合同文本 OCR 和解析
2. 风险条款自动识别
3. 法律问题提示
4. 修改建议生成
5. 标准模板对比
6. 合同版本管理

## 使用方式
```bash
/contract-review activate
/contract-review upload --file contract.pdf
/contract-review analyze --type employment
/contract-review compare --template standard-nda
```

## 适用场景
- 企业合同管理
- 法务部门
- 中小企业
- 个人合同审查
