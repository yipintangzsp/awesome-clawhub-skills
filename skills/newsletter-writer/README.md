# newsletter-writer

通讯写作（SkillPay 收费版）

## 分类
内容创作

## 价格
¥39/月

## 安装

```bash
clawhub install newsletter-writer
```

## 使用

```bash
newsletter-writer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/newsletter-writer.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
