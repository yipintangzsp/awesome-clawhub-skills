# Crypto Whale Alert Elite

精英版巨鲸追踪系统，监控链上大额资金动向，洞察市场趋势。

## 核心功能

### 1. 大额转账监控
- 实时追踪>$100K 的转账
- 多链支持（BTC/ETH/SOL/BNB 等）
- 自动识别交易所地址
- 机构钱包标记系统

### 2. 交易所流动分析
- 实时流入/流出统计
- 净流量趋势图表
- 历史对比分析
- 交易所储备变化

### 3. 智能分类
- 交易所热钱包/冷钱包
- 机构地址（MicroStrategy 等）
- 项目方国库
- 做市商钱包
- 未知巨鲸

### 4. 模式识别
- 累积 vs 派发
- 分批建仓检测
- 异常大额转账
- 关联地址分析

## 支持链
- Bitcoin
- Ethereum + L2s
- Solana
- BNB Chain
- Tron
- Polygon

## 警报类型
- 单笔大额转账
- 24h 累计流量异常
- 特定地址活动
- 交易所储备变化
- 稳定币增发/销毁
