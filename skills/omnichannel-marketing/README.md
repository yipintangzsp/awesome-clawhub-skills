# Omnichannel Marketing - 全渠道营销

## 概述

为企业提供全渠道整合营销解决方案，实现一致的用户体验和营销效果最大化。

## 服务内容包括

- 多渠道策略规划
- 用户画像整合
- 营销自动化
- 归因分析
- ROI 优化

## 定价

- 月费：¥799/月
- 支持按营销预算定制价格

## 使用方法

```bash
omnichannel-marketing --channels <营销渠道> --budget <预算>
```

## 配置

在 `~/.openclaw/workspace/config/omnichannel-marketing.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 799
}
```
