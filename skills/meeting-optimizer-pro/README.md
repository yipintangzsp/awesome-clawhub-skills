# Meeting Optimizer Pro - 会议优化专业版

## 概述

优化企业会议效率，从排期、议程到跟进全流程管理。

## 服务内容包括

- 智能会议排期
- 议程模板优化
- 会议纪要生成
- 行动项跟进
- 效率分析

## 定价

- 月费：¥399/月
- 支持按会议数量定制价格

## 使用方法

```bash
meeting-optimizer-pro --meetings <会议列表> --participants <参会人数>
```

## 配置

在 `~/.openclaw/workspace/config/meeting-optimizer-pro.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 399
}
```
