# 海外房产投资服务平台

## 技能描述
海外房产投资工具，全球房产数据、投资政策解读、税费计算、汇率风险、物业管理、退出渠道，一站式海外置业服务。

## 定价
- **标准版**: ¥799/月
- **专业版**: ¥1599/月
- **尊享版**: ¥2999/月（+ 顾问服务）

## 核心功能
1. 全球房产数据查询
2. 投资政策解读
3. 税费和成本计算
4. 汇率风险分析
5. 物业管理对接
6. 退出渠道规划

## 使用方式
```bash
/overseas-property activate
/overseas-property search --country Japan --city Tokyo --budget 500000
/overseas-property tax-calc --country USA --state CA --price 800000
/overseas-property risk-assess --country Thailand
```
