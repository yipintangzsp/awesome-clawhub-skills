# 抖音热点监控 🔥

实时监控抖音热点，追踪 trending 话题、音乐、挑战，帮你抓住流量风口。

## 功能特性

- 🔥 **热点榜单**: 实时抖音热榜、上升榜
- 🎵 **热门音乐**: trending BGM 追踪
- 💃 **挑战监控**: 热门挑战参与指南
- 💡 **创作灵感**: AI 生成脚本建议
- 🔔 **智能提醒**: 关键词监控推送

## 使用示例

### 查看实时热点
```
/douyin-hot-monitor --hot
```

### 监控特定话题
```
/douyin-hot-monitor --track "#新年挑战" --duration 24h
```

### 获取创作灵感
```
/douyin-hot-monitor --inspire --category 搞笑
```

### 设置关键词提醒
```
/douyin-hot-monitor --alert --keyword "美妆" --threshold 10000
```
