# douyin-hot-monitor - 抖音热点监控

## 描述
实时监控抖音热点榜单，追踪热门话题、音乐、挑战。提供创作灵感，蹭热点涨粉。

## 定价
- **包月订阅**: ¥29/月
- 实时热点推送
- 包含创作建议和脚本模板

## 用法
```bash
# 查看实时热点
/douyin-hot-monitor --hot

# 监控特定话题
/douyin-hot-monitor --track "#挑战"

# 获取创作灵感
/douyin-hot-monitor --inspire --category 剧情

# 设置提醒
/douyin-hot-monitor --alert --keyword 美妆
```
