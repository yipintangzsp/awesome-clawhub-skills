# Resource Allocation AI - 资源分配 AI

## 概述

智能优化企业资源分配，提升资源利用率和项目交付效率。

## 服务内容包括

- 资源盘点
- 需求匹配
- 优化分配
- 利用率分析
- 预测规划

## 定价

- 月费：¥499/月
- 支持按资源规模定制价格

## 使用方法

```bash
resource-allocation-ai --resources <资源列表> --projects <项目列表>
```

## 配置

在 `~/.openclaw/workspace/config/resource-allocation-ai.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 499
}
```
