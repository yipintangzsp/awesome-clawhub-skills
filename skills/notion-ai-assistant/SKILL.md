# notion-ai-assistant

Notion AI 助手（SkillPay 收费版）

## 使用方式

```bash
notion-ai-assistant --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/notion-ai-assistant.json` 配置 SkillPay API Key。
