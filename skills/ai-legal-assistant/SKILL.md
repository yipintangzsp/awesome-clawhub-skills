# ai-legal-assistant - AI 智能法律助手系统

**痛点**：合同审查耗时，法律风险难识别，律师费用高，法务响应慢
**价值**：AI 快速审查合同，识别风险条款，降低法律风险 70%+

## 使用方式

```bash
ai-legal-assistant --help
```

## 功能特点

- ⚡ **合同智能审查** - AI 自动识别风险条款、霸王条款、漏洞
- 📊 **法律文书生成** - 基于模板自动生成合同、协议、函件等
- 💰 **合规风险评估** - 评估业务合规性，预警潜在法律风险
- 🔒 **案例智能检索** - 快速检索相关判例，支持法律决策
- 🎯 **法律咨询辅助** - 常见问题 AI 解答，复杂问题转接律师

## 适用场景

- 企业合同审查
- 法律文书起草
- 合规风险管理
- 日常法律咨询

## 配置

在 `~/.openclaw/workspace/config/ai-legal-assistant.json` 配置 SkillPay API Key。
