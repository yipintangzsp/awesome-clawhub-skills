# ebook-formatter

电子书格式化（SkillPay 收费版）

## 使用方式

```bash
ebook-formatter --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/ebook-formatter.json` 配置 SkillPay API Key。
