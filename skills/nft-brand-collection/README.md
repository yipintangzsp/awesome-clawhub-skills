# NFT Brand Collection - NFT 品牌系列

## 概述

为企业提供 NFT 品牌系列创建服务，从设计到铸造到上架一站式服务。

## 服务内容包括

- 品牌 NFT 概念设计
- 艺术素材生成
- 智能合约开发
- 批量铸造部署
- 市场上架指导

## 定价

- 单次：¥999/次
- 支持按系列规模定制价格

## 使用方法

```bash
nft-brand-collection --brand <品牌名> --size <系列数量>
```

## 配置

在 `~/.openclaw/workspace/config/nft-brand-collection.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_item": 999
}
```
