# Crypto Tax Enterprise - 企业加密税务

## 概述

为企业提供加密货币税务合规解决方案，自动化交易导入和税务计算。

## 服务内容包括

- 多链交易导入
- 成本基础计算
- 盈亏分析报告
- 税务报表生成
- 审计支持文档

## 定价

- 月费：¥699/月
- 支持按交易量定制价格

## 使用方法

```bash
crypto-tax-enterprise --entity <企业实体> --chains <区块链>
```

## 配置

在 `~/.openclaw/workspace/config/crypto-tax-enterprise.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 699
}
```
