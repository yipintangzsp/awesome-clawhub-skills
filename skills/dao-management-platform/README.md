# DAO Management Platform - DAO 管理平台

## 概述

为去中心化自治组织提供完整的治理和运营管理工具。

## 服务内容包括

- 提案创建与投票
- 财库资金管理
- 成员治理权限
- 链上执行自动化
- 治理数据分析

## 定价

- 月费：¥699/月
- 支持按 DAO 规模定制价格

## 使用方法

```bash
dao-management-platform --dao <DAO 名称> --members <成员数>
```

## 配置

在 `~/.openclaw/workspace/config/dao-management-platform.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 699
}
```
