# 女性健康与经期管理系统

## 技能描述
女性专属健康管理，经期追踪、排卵期预测、备孕指导、孕期管理、更年期支持，全周期女性健康关怀。

## 定价
- **标准版**: ¥299/月
- **备孕版**: ¥599/月
- **孕期版**: ¥799/月

## 核心功能
1. 经期自动追踪和预测
2. 排卵期精准计算
3. 备孕指导和提醒
4. 孕期周数管理
5. 症状记录和缓解
6. 健康建议推送

## 使用方式
```bash
/female-health activate
/female-health track --cycle 28 --period_start 2024-03-01
/female-health predict --next_period
/female-health fertility --optimize true
```

- 经期管理

- 孕期妈妈
- 更年期女性
