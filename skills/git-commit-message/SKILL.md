# Git Commit Message

Git commit 生成工具 - 根据代码变更自动生成规范的 commit 信息

## 使用方式

```bash
git-commit-message --diff "代码变更内容"
```

## 功能特点

- 📝 自动生成 commit
- 🎯 遵循约定式提交
- 📊 变更分析
- 💰 ¥9/月订阅

## 配置

在 `~/.openclaw/workspace/config/git-commit-message.json` 配置 SkillPay API Key。
