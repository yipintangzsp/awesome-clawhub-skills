# Content Calendar Pro - 内容日历专业版

## 概述

为企业提供智能内容日历规划，优化内容发布策略，提升传播效果。

## 服务内容包括

- 内容主题规划
- 发布排期优化
- AI 内容建议
- 热点追踪
- 效果预测

## 定价

- 月费：¥399/月
- 支持按内容量定制价格

## 使用方法

```bash
content-calendar-pro --channels <发布渠道> --frequency <发布频率>
```

## 配置

在 `~/.openclaw/workspace/config/content-calendar-pro.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 399
}
```
