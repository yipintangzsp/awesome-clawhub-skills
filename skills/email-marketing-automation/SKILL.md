# email-marketing-automation

邮件营销自动化（¥79/月）

## 使用方式

```bash
email-marketing-automation --help
```

## 功能特点

- 📧 邮件序列设计
- 🎯 受众细分
- ⏰ 发送时间优化
- 📊 打开率分析
- 🔄 A/B 测试自动化
- 🔒 SkillPay 收费集成

## 配置

在 `~/.openclaw/workspace/config/email-marketing-automation.json` 配置 SkillPay API Key 和邮件平台。
