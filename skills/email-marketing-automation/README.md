# email-marketing-automation

邮件营销自动化（¥79/月）

## 分类
营销/邮件营销

## 价格
¥79/月

## 安装

```bash
clawhub install email-marketing-automation
```

## 使用

```bash
email-marketing-automation [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/email-marketing-automation.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 79,
  "email_platform": "mailchimp",
  "campaign_type": "nurture",
  "send_frequency": "weekly"
}
```

## 功能

- 自动化邮件序列设计
- 智能受众细分

- 主题行 A/B 测试
- 用户行为触发邮件
- 营销漏斗追踪
