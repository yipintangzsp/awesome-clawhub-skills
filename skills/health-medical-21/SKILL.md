# 个人健康数据整合与 AI 健康顾问系统

## 技能描述
个人健康数据中枢，整合智能手表、体检报告、医疗记录，AI 分析健康趋势、提供个性化健康建议、疾病风险预警、就医指导。

## 定价
- **标准版**: ¥499/月
- **专业版**: ¥999/月（+ 深度分析）
- **家庭版**: ¥1499/月（5 人）

## 核心功能
1. 多设备健康数据同步
2. 体检报告 AI 解读
3. 健康趋势分析和预警
4. 个性化健康建议
5. 疾病风险评估
6. 就医科室推荐

## 使用方式
```bash
/health-advisor activate
/health-advisor sync --devices apple-watch,huawei
/health-advisor report --upload checkup.pdf
/health-advisor ask --symptom "头痛、乏力"
```

## 适用场景
- 个人健康管理
- 慢病患者监测
- 亚健康人群改善
- 家庭健康监护
