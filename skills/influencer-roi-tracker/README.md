# Influencer ROI Tracker - 网红 ROI 追踪

## 概述

追踪和分析 KOL/网红营销活动的投资回报率，优化营销预算分配。

## 服务内容包括

- KOL 效果追踪
- ROI 计算分析
- 转化归因
- 成本核算
- 优化建议

## 定价

- 月费：¥499/月
- 支持按活动数量定制价格

## 使用方法

```bash
influencer-roi-tracker --campaigns <活动列表> --platforms <平台>
```

## 配置

在 `~/.openclaw/workspace/config/influencer-roi-tracker.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 499
}
```
