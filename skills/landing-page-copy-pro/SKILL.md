# landing-page-copy-pro - 落地页文案专业版

## 技能描述
专业版落地页文案生成工具，基于转化心理学和 AIDA 模型，生成高转化落地页文案。支持多行业模板、A/B 测试版本、SEO 优化。

## 功能特性
- 🎯 AIDA 模型文案结构
- 📝 多行业模板库
- ✨  headline 优化
- 💪 行动号召（CTA）优化
- 🔍 SEO 关键词优化
- 🧪 A/B 测试版本生成

## 使用方式
```bash
# 生成落地页文案
/landing-page-copy-pro --generate --product <产品名> --industry <行业>

# 优化 headline
/landing-page-copy-pro --headline --topic <主题> --count 10

# 生成 CTA
/landing-page-copy-pro --cta --goal <转化目标>

# A/B 测试版本
/landing-page-copy-pro --abtest --page <URL>
```
