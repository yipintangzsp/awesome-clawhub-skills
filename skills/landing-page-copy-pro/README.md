## 核心框架

### AIDA 模型
- **Attention**（注意）：吸引眼球的 headline
- **Interest**（兴趣）：激发兴趣的卖点
- **Desire**（欲望）：创造购买欲望
- **Action**（行动）：清晰的 CTA

### PAS 模型
- **Problem**（问题）：指出痛点
- **Agitation**（煽动）：放大痛苦
- **Solution**（解决）：提供方案

### FAB 模型
- **Feature**（特性）：产品功能
- **Advantage**（优势）：相比竞品
- **Benefit**（利益）：用户收益
