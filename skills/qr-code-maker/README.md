## 安装

```bash
clawhub install qr-code-maker
```

## 使用

```bash
qr-code-maker [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/qr-code-maker.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 2
}
```
