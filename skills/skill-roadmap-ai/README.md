技能路线图 AI（¥8/次）

## 分类
学习成长

## 价格
¥8/次

## 安装

```bash
clawhub install skill-roadmap-ai
```

## 使用

```bash
skill-roadmap-ai [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/skill-roadmap-ai.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 8
}
```

- 自动收费验证

- 详细日志
