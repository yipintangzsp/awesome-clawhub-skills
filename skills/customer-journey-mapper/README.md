# Customer Journey Mapper - 客户旅程映射

## 概述

帮助企业绘制和分析客户旅程地图，优化用户体验，提升转化率。

## 服务内容包括

- 用户画像定义
- 旅程地图绘制
- 触点分析
- 痛点识别
- 优化建议

## 定价

- 月费：¥599/月
- 支持按旅程复杂度定制价格

## 使用方法

```bash
customer-journey-mapper --persona <用户画像> --touchpoints <触点>
```

## 配置

在 `~/.openclaw/workspace/config/customer-journey-mapper.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
