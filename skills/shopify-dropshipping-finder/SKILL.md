# shopify-dropshipping-finder

Shopify 代发货选品（¥79/月）

## 使用方式

```bash
shopify-dropshipping-finder --help
```

## 功能特点

- 🔥 热销产品发现
- 📊 市场趋势分析
- 💰 利润计算器
- 🏪 供应商匹配
- 📈 竞品监控
- 🔒 SkillPay 收费集成

## 配置

在 `~/.openclaw/workspace/config/shopify-dropshipping-finder.json` 配置 SkillPay API Key 和选品参数。
