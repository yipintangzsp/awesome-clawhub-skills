# shopify-dropshipping-finder

Shopify 代发货选品（¥79/月）

## 分类
电商/选品工具

## 价格
¥79/月

## 安装

```bash
clawhub install shopify-dropshipping-finder
```

## 使用

```bash
shopify-dropshipping-finder [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/shopify-dropshipping-finder.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 79,
  "niche": "electronics",
  "min_profit_margin": 30,
  "supplier_platforms": ["aliexpress", "cjdropshipping"]
}
```

## 功能

- 热销产品智能发现
- 市场趋势实时分析

- 竞品店铺监控
- 产品上架建议
- 季节性选品推荐
