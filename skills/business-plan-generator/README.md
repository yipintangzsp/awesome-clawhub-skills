生产力/商业工具

## 价格
¥199/次

## 安装

```bash
clawhub install business-plan-generator
```

## 使用

```bash
business-plan-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/business-plan-generator.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 199,
  "industry": "technology",
  "business_type": "startup",
  "target_market": "global"
}
```
