# competitor-price-monitor - 竞品价格智能监控系统

**痛点**：竞品价格变化快，手动追踪耗时，调价不及时丢失订单，价格战被动
**价值**：7x24 小时自动监控，智能调价建议，保持价格竞争力

## 使用方式

```bash
competitor-price-monitor --help
```

## 功能特点

- ⚡ **实时价格监控** - 自动追踪竞品价格变化，分钟级更新
- 📊 **价格趋势分析** - 历史价格数据可视化，识别定价规律
- 💰 **自动调价建议** - 基于成本和竞品价格，生成最优定价建议
- 🔒 **市场份额追踪** - 分析价格与销量关系，评估市场地位
- 🎯 **促销情报收集** - 监控竞品促销活动，及时应对

## 适用场景

- 亚马逊/Shopify 价格监控
- 动态定价策略制定
- 竞品情报分析
- 价格战应对策略

## 配置

在 `~/.openclaw/workspace/config/competitor-price-monitor.json` 配置 SkillPay API Key。
