# Code Comment Writer

代码注释生成工具 - 自动为代码添加清晰的注释

## 使用方式

```bash
code-comment-writer --file path/to/code.js --style jsdoc
```

## 功能特点

- 📝 自动注释
- 🎯 多风格支持
- 💡 函数说明
- 💰 ¥9/次

## 配置

在 `~/.openclaw/workspace/config/code-comment-writer.json` 配置 SkillPay API Key。
