# YouTube Thumbnail AI - YouTube 缩略图 AI

## Description
YouTube 缩略图 AI 设计工具，基于视频内容自动生成高点击率缩略图。运用色彩心理学、人脸表情、文字排版等原理，提升视频 CTR。

## Usage
```bash
# 生成缩略图
/youtube-thumbnail-ai --title "AI 工具教程" --style "minimal" --emotion "surprised"

# A/B 测试
/youtube-thumbnail-ai --action "ab-test" --variants 3 --title "视频标题"

# 分析竞品
/youtube-thumbnail-ai --action "analyze" --video-url "youtube.com/watch?v=xxx"
```

### 参数说明
- `--title`: 视频标题
- `--style`: 设计风格 (minimal, bold, colorful, dark, bright)
- `--emotion`: 表情类型 (surprised, happy, serious, excited)
- `--action`: 操作类型 (generate, ab-test, analyze, optimize)
- `--video-url`: 竞品视频链接
