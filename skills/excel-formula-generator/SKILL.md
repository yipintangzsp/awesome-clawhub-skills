# excel-formula-generator

Excel 公式生成（SkillPay 收费版）

## 使用方式

```bash
excel-formula-generator --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/excel-formula-generator.json` 配置 SkillPay API Key。
