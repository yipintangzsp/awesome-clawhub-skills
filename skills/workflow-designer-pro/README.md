# Workflow Designer Pro - 工作流设计专业版

## 概述

为企业设计和优化业务流程，实现自动化和效率提升。

## 服务内容包括

- 流程梳理分析
- 自动化设计
- 工具选型建议
- 实施指导
- 效果评估

## 定价

- 单次：¥699/次
- 支持按流程复杂度定制价格

## 使用方法

```bash
workflow-designer-pro --process <业务流程> --team <团队规模>
```

## 配置

在 `~/.openclaw/workspace/config/workflow-designer-pro.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_project": 699
}
```
