# 智能税务筹划与优化系统

## 技能描述
个人和企业税务筹划工具，自动计算最优税务方案、税收优惠政策匹配、跨境税务规划、税务风险识别，合法降低税负。

## 定价
- **个人版**: ¥599/月
- **企业版**: ¥1599/月
- **跨境版**: ¥2999/月

## 核心功能
1. 个税自动计算和优化
2. 企业所得税筹划
3. 税收优惠政策匹配
4. 跨境税务规划
5. 税务风险识别
6. 税务申报辅助

## 使用方式
```bash
/tax-optimization activate
/tax-optimization personal --income 500000 --deductions auto
/tax-optimization corporate --profit 2000000
/tax-optimization cross-border --countries CN,US,HK
```

## 适用场景
- 高净值个人税务规划
- 中小企业税务筹划
- 跨境电商税务
- 外籍人士税务
