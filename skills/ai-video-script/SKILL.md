# AI Video Script - AI 视频脚本生成

## Description
专业视频脚本生成工具，适用于 YouTube、B 站、抖音、小红书等平台。自动生成开头钩子、内容结构、结尾 CTA，提升视频完播率和互动率。

## Usage
```bash
# 生成视频脚本
/ai-video-script --platform "youtube" --topic "AI 工具教程" --duration 600

# 指定风格
/ai-video-script --platform "tiktok" --style "humorous" --duration 60

# 生成系列脚本
/ai-video-script --series --episodes 5 --topic "跨境电商入门"
```

### 参数说明
- `--platform`: 视频平台 (youtube, bilibili, tiktok, douyin, xiaohongshu)
- `--topic`: 视频主题
- `--duration`: 视频时长 (秒)
- `--style`: 风格 (professional, humorous, educational, storytelling)
- `--series`: 是否生成系列脚本
