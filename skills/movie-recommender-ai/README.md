# movie-recommender-ai

电影推荐 AI（¥3/次）

## 分类
娱乐休闲

## 价格
¥3/次

## 安装

```bash
clawhub install movie-recommender-ai
```

## 使用

```bash
movie-recommender-ai [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/movie-recommender-ai.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 3
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
