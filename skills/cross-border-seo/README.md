# Cross Border SEO - 跨境 SEO

## 概述

为跨境电商提供多语言、多区域的 SEO 优化服务，提升全球搜索排名。

## 服务内容包括

- 多语言关键词研究
- 本地化内容优化
- 多区域网站结构
- 外链建设
- 排名追踪报告

## 定价

- 月费：¥599/月
- 支持按市场数量定制价格

## 使用方法

```bash
cross-border-seo --markets <目标市场> --languages <语言>
```

## 配置

在 `~/.openclaw/workspace/config/cross-border-seo.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
