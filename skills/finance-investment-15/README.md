# 全球宏观经济数据监控与投资信号系统

## 产品概述

专业宏观经济分析平台，实时追踪全球主要经济体数据，AI 解读政策影响，为投资决策提供宏观视角支持。

## 监控范围

### 经济指标
- GDP、CPI、PPI
- 就业数据、PMI
- 零售销售、工业产出
- 贸易数据、外汇储备

### 央行政策
- 利率决议
- 量化宽松
- 前瞻指引
- 官员讲话

### 市场影响
- 股债汇商联动分析
- 风险偏好监测
- 资金流向追踪
- 估值变化

## 核心功能

### 数据追踪
- 自动抓取官方数据
- 预期值 vs 实际值
- 历史数据对比
- 修订值追踪

### AI 解读
- 数据影响分析
- 政策走向预测
- 市场反应预判
- 投资含义提炼

### 资产配置
- 战略资产配置
- 战术调整建议
- 风险平价模型
- 情景分析
