# alibaba-quality-inspector - 阿里巴巴智能质检系统

**痛点**：供应商质量参差不齐，样品与大货不符，采购踩坑损失大
**价值**：AI 评估供应商资质，提前识别风险，降低 80% 采购损失

## 使用方式

```bash
alibaba-quality-inspector --help
```

## 功能特点

- ⚡ **供应商资质审核** - 自动核查工厂认证、年限、交易记录
- 📊 **质量风险评估** - 基于历史投诉、退货率评估质量风险
- 💰 **价格合理性分析** - 对比市场均价，识别低价陷阱
- 🔒 **验厂报告生成** - 自动生成验厂检查清单和风险评估报告
- 🎯 **备选供应商推荐** - 基于质量评分推荐可靠备选供应商

## 适用场景

- 1688/阿里巴巴采购前评估
- 供应商资质审核
- 大货生产前风险控制
- 供应链管理优化

## 配置

在 `~/.openclaw/workspace/config/alibaba-quality-inspector.json` 配置 SkillPay API Key。
