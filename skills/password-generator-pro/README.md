## 安装

```bash
clawhub install password-generator-pro
```

## 使用

```bash
password-generator-pro [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/password-generator-pro.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 2
}
```
