# 智能法律文书生成系统

## 产品概述

法律文书自动化生成平台，整合海量法律文书模板，AI 根据案情自动生成专业法律文书，提升律师工作效率。

## 文书类型

### 诉讼文书
- 起诉状
- 答辩状
- 上诉状
- 代理词

### 合同文书
- 各类合同
- 补充协议
- 变更协议

### 函件文书
- 律师函
- 催告函
- 告知函

### 其他文书
- 遗嘱
- 授权委托书
- 法律意见书

### 模板管理
- 海量模板库

- 模板更新

- 案情信息提取
- 自动填充内容

### 质量检查
