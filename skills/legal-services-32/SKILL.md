# 智能法律文书生成系统

## 技能描述
法律文书自动化工具，基于模板和案情自动生成起诉状、答辩状、合同、律师函等法律文书，节省律师时间，降低文书成本。

## 定价
- **标准版**: ¥699/月（30 份）
- **专业版**: ¥1499/月（100 份）
- **律所版**: ¥2999/月（无限 + 定制模板）

## 核心功能
1. 文书模板库
2. 案情信息提取
3. 文书自动生成
4. 格式自动调整
5. 法律引用校验
6. 文书质量检查

## 使用方式
```bash
/legal-doc-gen activate
/legal-doc-gen create --type complaint --case case123
/legal-doc-gen template --search 劳动合同
/legal-doc-gen validate --document DOC456
```
