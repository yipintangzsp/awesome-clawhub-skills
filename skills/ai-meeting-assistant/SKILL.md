## 使用方式

```bash
# 基础用法
ai-meeting-assistant --transcript path/to/transcript.txt

# 高级用法
ai-meeting-assistant --transcript meeting.txt --format markdown --action-items --summary
```

## 功能特点

- ⚡ **AI 自动纪要生成** - 语音/文字转结构化纪要，3 分钟出稿，准确率 95%+
- ✅ **智能待办提取** - 自动识别任务、负责人、截止时间，生成待办清单
- 👥 **发言人识别** - 区分不同发言人，标注关键观点和决策
- 📊 **会议洞察分析** - 统计发言时长、话题分布、决策效率，优化会议质量
- 🔔 **待办自动追踪** - 与飞书/钉钉/企微集成，自动分派和提醒待办事项

## 适用场景

- 🏢 **企业例会** - 周会/月会/季度会自动生成纪要，存档可追溯
- 🤝 **客户会议** - 销售/需求讨论自动记录，避免遗漏关键信息
- 💻 **远程协作** - 跨时区会议自动整理，异步协作更高效
- 📋 **面试记录** - HR 面试自动记录评估要点，人才库自动归档
