# financial-model-builder

财务模型构建（¥149/次）

## 分类
生产力/财务工具

## 价格
¥149/次

## 安装

```bash
clawhub install financial-model-builder
```

## 使用

```bash
financial-model-builder [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/financial-model-builder.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 149,
  "model_type": "startup",
  "forecast_years": 5,
  "currency": "CNY"
}
```

## 功能

- 利润表/资产负债表/现金流量表联动
- 5 年财务预测
- DCF/可比公司估值
- 敏感性分析
- 多情景模拟（乐观/中性/悲观）
- 关键财务比率分析
- Excel 导出功能
