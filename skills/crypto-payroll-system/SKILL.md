# Crypto Payroll System

加密薪资系统 - 企业加密货币薪资发放

## 使用方式

```bash
crypto-payroll-system --employees <员工数> --tokens <代币列表>
```

## 功能特点

- 💰 多币种薪资发放
- 📊 自动汇率转换
- 📋 合规报表生成
- 🔔 发放通知
- 💰 ¥599/月订阅

## 配置

在 `~/.openclaw/workspace/config/crypto-payroll-system.json` 配置 SkillPay API Key。
