# Shopee Price Tracker

Shopee 价格追踪工具 - 监控竞品价格变化和历史趋势

## 使用方式

```bash
shopee-price-tracker --product-id 12345678
```

## 功能特点

- 📊 价格历史追踪
- 🔔 价格提醒
- 📈 趋势分析
- 💰 ¥19/月订阅

## 配置

在 `~/.openclaw/workspace/config/shopee-price-tracker.json` 配置 SkillPay API Key。
