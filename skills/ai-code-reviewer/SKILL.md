**价值**：AI 自动审查代码，秒级发现问题，减少 70% Bug，提升代码质量

## 使用方式

```bash
# 基础用法
ai-code-reviewer --file path/to/code.js

# 高级用法
ai-code-reviewer --dir ./src --language javascript --strict --output report.md
```

## 功能特点

- ⚡ **智能 Bug 检测** - 识别空指针、内存泄漏、逻辑错误等常见 Bug
- 🔒 **安全漏洞扫描** - SQL 注入、XSS、CSRF 等安全问题自动预警
- 💡 **代码优化建议** - 性能优化、代码重构、最佳实践建议
- 📊 **质量评分报告** - 多维度评分（可维护性/性能/安全），生成改进清单
- 🔗 **CI/CD 集成** - GitHub/GitLab 自动集成，提交前自动审查

## 适用场景

- 💻 **日常开发** - 提交前自动审查，避免低级错误
- 🔄 **代码重构** - 识别技术债务，给出重构优先级建议
- 👨‍🏫 **团队培训** - 新人代码自动 review，快速提升代码水平
- 🔐 **安全审计** - 上线前安全扫描，降低安全风险
