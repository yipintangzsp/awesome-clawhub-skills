# testimonial-generator

testimonial-generator - 实用工具（SkillPay 收费版）

## 使用方式

```bash
testimonial-generator --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/testimonial-generator.json` 配置 SkillPay API Key。
