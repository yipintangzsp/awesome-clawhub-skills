# blockchain-forensics

区块链取证分析（¥199/月）

## 使用方式

```bash
blockchain-forensics --help
```

## 功能特点

- 🔍 链上资金追踪
- 🕵️ 地址关联分析
- 📊 交易图谱可视化
- ⚠️ 风险地址识别
- 🔗 跨链追踪
- 🔒 SkillPay 收费集成

## 配置

在 `~/.openclaw/workspace/config/blockchain-forensics.json` 配置 SkillPay API Key 和分析参数。
