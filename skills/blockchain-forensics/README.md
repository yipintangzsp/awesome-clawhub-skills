# blockchain-forensics

区块链取证分析（¥199/月）

## 分类
加密货币/安全分析

## 价格
¥199/月

## 安装

```bash
clawhub install blockchain-forensics
```

## 使用

```bash
blockchain-forensics [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/blockchain-forensics.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 199,
  "chains": ["ethereum", "bsc", "tron"],
  "risk_threshold": 0.7
}
```

## 功能

- 链上资金流向追踪
- 地址关联图谱分析
- 风险地址智能识别
- 黑客/洗钱地址库比对
- 跨链资金追踪
- 可视化交易图谱
- 合规报告生成
