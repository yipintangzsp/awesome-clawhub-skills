# hr-recruitment-assistant

HR 招聘助手（¥79/月）

## 分类
生产力/人力资源

## 价格
¥79/月

## 安装

```bash
clawhub install hr-recruitment-assistant
```

## 使用

```bash
hr-recruitment-assistant [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/hr-recruitment-assistant.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 79,
  "industry": "technology",
  "role_level": "mid-senior",
  "remote_option": true
}
```

## 功能

- 职位描述（JD）智能生成
- 简历关键词筛选
- 结构化面试问题
- 候选人能力评分
- 背景调查清单
- Offer 邮件模板
- 薪酬建议参考
- 招聘流程管理
