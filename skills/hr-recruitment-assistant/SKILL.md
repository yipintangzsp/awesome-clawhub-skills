# hr-recruitment-assistant

HR 招聘助手（¥79/月）

## 使用方式

```bash
hr-recruitment-assistant --help
```

## 功能特点

- 📝 JD 智能生成
- 📄 简历筛选
- 💬 面试问题生成
- 📊 候选人评分
- 📧 Offer 邮件模板
- 🔒 SkillPay 收费集成

## 配置

在 `~/.openclaw/workspace/config/hr-recruitment-assistant.json` 配置 SkillPay API Key 和招聘参数。
