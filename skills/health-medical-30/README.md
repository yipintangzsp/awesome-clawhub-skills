# 医疗资源智能匹配与就医助手系统

## 产品概述

智能就医服务平台，帮助用户快速找到合适的医院和医生，提供就诊全流程指导，让看病不再难。

## 核心功能

### 症状自查
- AI 初步诊断
- 可能疾病列表
- 紧急程度评估
- 就医建议

### 医生匹配
- 科室推荐
- 医生擅长匹配
- 患者评价参考
- 号源情况

### 医院对比
- 医院等级
- 专科排名
- 距离和交通
- 费用对比

### 就诊辅助
- 挂号提醒
- 就诊准备清单
- 问题清单生成
- 医嘱整理

### 病历管理
- 电子病历存储
- 检查报告整理
- 用药记录
- 过敏史管理
