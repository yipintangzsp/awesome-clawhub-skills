# 医疗资源智能匹配与就医助手系统

## 技能描述
智能就医平台，症状自查、科室推荐、医院医生匹配、挂号提醒、就诊准备、病历管理，让就医更高效。

- **专业版**: ¥599/月（+ 绿色通道）
- **尊享版**: ¥999/月（+ 陪诊服务）

## 核心功能
1. AI 症状自查
2. 科室和医生推荐
3. 医院匹配和对比
4. 挂号提醒和辅助
5. 就诊准备清单
6. 电子病历管理

## 使用方式
```bash
/medical-assistant activate
/medical-assistant symptom-check --symptoms "头痛、发热"
/medical-assistant find-doctor --department neurology --city Beijing
/medical-assistant prepare --appointment APT123
```

## 适用场景
- 不确定挂什么科
- 寻找合适医生
- 就医流程指导
- 病历整理
