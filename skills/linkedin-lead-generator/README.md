# linkedin-lead-generator

LinkedIn 潜客生成（¥129/月）

## 分类
营销/潜客开发

## 价格
¥129/月

## 安装

```bash
clawhub install linkedin-lead-generator
```

## 使用

```bash
linkedin-lead-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/linkedin-lead-generator.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 129,
  "target_industry": "technology",
  "company_size": "50-500",
  "decision_maker_level": ["CEO", "CTO", "VP"]
}
```

## 功能

- 精准目标客户筛选
- AI 外展消息生成
- 客户画像深度分析
- 二度人脉关系挖掘
- 互动转化率追踪
- 自动化跟进流程
- CRM 数据导出
