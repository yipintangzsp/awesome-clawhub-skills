# linkedin-lead-generator

LinkedIn 潜客生成（¥129/月）

## 使用方式

```bash
linkedin-lead-generator --help
```

## 功能特点

- 🔍 精准客户筛选
- 📧 自动外展消息
- 📊 客户画像分析
- 🤝 关系链挖掘
- 📈 转化率追踪
- 🔒 SkillPay 收费集成

## 配置

在 `~/.openclaw/workspace/config/linkedin-lead-generator.json` 配置 SkillPay API Key 和 LinkedIn 参数。
