# legal-contract-reviewer

法律合同审查（¥99/次）

## 分类
生产力/法律工具

## 价格
¥99/次

## 安装

```bash
clawhub install legal-contract-reviewer
```

## 使用

```bash
legal-contract-reviewer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/legal-contract-reviewer.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 99,
  "contract_type": "service",
  "jurisdiction": "CN"
}
```

## 功能

- 合同条款智能分析
- 潜在风险点识别
- 不利条款标记
- 修改建议生成
- 法律合规性检查
- 风险评分报告
- 常见合同模板对比
- 关键条款摘要

⚠️ 免责声明：本工具仅供参考，不构成法律意见。重大合同请咨询专业律师。
