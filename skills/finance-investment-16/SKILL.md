# 企业财务健康度分析与风险预警系统

## 技能描述
企业财务分析工具，自动抓取财报数据、计算财务比率、评估偿债能力、盈利能力、运营效率、成长能力，识别财务风险，支持投资决策和信贷审批。

## 定价
- **标准版**: ¥699/月（100 企业）
- **专业版**: ¥1499/月（500 企业）
- **机构版**: ¥2999/月（无限 + 定制模型）

## 核心功能
1. 财报数据自动抓取和解析
2. 财务比率自动计算
3. 杜邦分析和趋势分析
4. 同业对比和排名
5. 财务风险预警
6. 信用评分模型

## 使用方式
```bash
/financial-analysis activate
/financial-analysis analyze --company 00700.HK
/financial-analysis compare --industry tech
/financial-analysis risk-check --portfolio holdings.csv
```

## 适用场景
- 股票投资研究
- 信贷审批
- 供应链管理
- 并购尽职调查
