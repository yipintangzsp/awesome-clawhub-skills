# 企业财务健康度分析与风险预警系统

## 产品概述

专业企业财务分析平台，自动化财报分析，多维度评估企业财务健康度，提前识别财务风险，支持投资和信贷决策。

## 核心功能

### 财报分析
- 三表自动解析
- 关键指标提取
- 同比环比分析
- 会计质量评估

### 财务比率
- 偿债能力：流动比率、速动比率、资产负债率
- 盈利能力：ROE、ROA、毛利率、净利率
- 运营效率：存货周转、应收周转、总资产周转
- 成长能力：收入增长、利润增长、资产增长

### 杜邦分析
- ROE 拆解
- 驱动因素识别
- 趋势分析
- 同业对比

### 风险预警
- 财务造假识别
- 流动性风险
- 偿债风险
- 经营恶化预警

### 信用评分
- 内部评级模型
- 违约概率计算
- 信用利差分析
- 评级迁移追踪

## 数据覆盖
- A 股、港股、美股上市公司
- 债券发行人
- 非上市企业（专业版）
