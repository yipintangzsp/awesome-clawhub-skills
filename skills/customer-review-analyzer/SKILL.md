# customer-review-analyzer - 客户评论智能分析系统

**痛点**：评论太多看不过来，问题分散难汇总，改进方向不明确，客户流失不知原因
**价值**：AI 自动分析海量评论，提炼核心问题，指导产品改进提升复购

## 使用方式

```bash
customer-review-analyzer --help
```

## 功能特点

- ⚡ **情感自动识别** - AI 分析评论情感，识别满意/不满客户
- 📊 **问题归类汇总** - 自动归类质量问题、物流问题、服务问题等
- 💰 **竞品评论对比** - 对比竞品评论，找出自身优势和劣势
- 🔒 **风险预警系统** - 识别集中负面评价，预警潜在危机
- 🎯 **产品改进建议** - 基于评论数据生成具体产品优化建议

## 适用场景

- 亚马逊/Shopify 评论分析
- 产品改进决策支持
- 客户满意度监控
- 品牌声誉管理

## 配置

在 `~/.openclaw/workspace/config/customer-review-analyzer.json` 配置 SkillPay API Key。
