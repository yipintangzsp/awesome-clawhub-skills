# Global Trend Analyzer - 全球趋势分析

## 概述

为跨境电商提供全球市场趋势分析，帮助发现热销产品和市场机会。

## 服务内容包括

- 市场趋势分析
- 热销产品发现
- 竞品监控
- 价格趋势追踪
- 选品建议报告

## 定价

- 月费：¥599/月
- 支持按分析类目定制价格

## 使用方法

```bash
global-trend-analyzer --category <商品类别> --markets <目标市场>
```

## 配置

在 `~/.openclaw/workspace/config/global-trend-analyzer.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
