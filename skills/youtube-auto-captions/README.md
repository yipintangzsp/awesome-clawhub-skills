# YouTube Auto Captions 📺

自动为 YouTube 视频生成高质量字幕，支持多语言翻译，提升视频 SEO 和可访问性。

## 功能特性

- 🎯 **自动识别**: AI 语音识别，准确率 95%+
- 🌍 **多语言**: 支持 50+ 语言翻译
- ⏱️ **时间轴**: 自动校准字幕时间
- 📝 **多格式**: SRT, VTT, TXT 导出
- 🔍 **SEO 优化**: 字幕助力视频搜索排名

## 使用示例

### 生成中文字幕
```
/youtube-auto-captions --video dQw4w9WgXcQ --lang zh
```

### 多语言翻译
```
/youtube-auto-captions --video dQw4w9WgXcQ --translate en,ja,ko,zh
```

### 批量处理播放列表
```
/youtube-auto-captions --playlist PLxxx --lang zh
```

### 导出字幕文件
```
/youtube-auto-captions --video dQw4w9WgXcQ --export srt --output subtitles.srt
```
