## 使用方式

```bash
# 基础用法
ai-inference-optimizer --model <模型路径> --target <部署环境>

# 高级用法
ai-inference-optimizer --model bert-base --target gpu --quantize int8 --batch auto
```

## 功能特点

- ⚡ **智能推理加速** - 自动选择最优推理引擎（TensorRT/OpenVINO/ONNX）
- 📦 **模型量化压缩** - FP32→INT8 量化，模型缩小 4 倍，速度提升 3 倍
- 🔧 **批处理优化** - 动态批处理策略，最大化 GPU 利用率
- 📊 **性能基准测试** - 多维度性能对比，给出最优配置建议
- 🎯 **硬件适配优化** - GPU/CPU/边缘设备自动适配，发挥最佳性能
