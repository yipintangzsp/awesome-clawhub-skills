# ai-hr-recruiter - AI 智能招聘官系统

**痛点**：简历太多筛不过来，人岗匹配难判断，面试效率低，招聘周期长
**价值**：AI 自动筛选简历，精准人岗匹配，缩短招聘周期 50%+

## 使用方式

```bash
ai-hr-recruiter --help
```

## 功能特点

- ⚡ **简历智能筛选** - AI 自动解析简历，匹配岗位要求初筛候选人
- 📊 **人岗匹配分析** - 多维度评估候选人匹配度，给出推荐评分
- 💰 **面试评估辅助** - 基于简历生成面试问题，评估面试表现
- 🔒 **人才库管理** - 自动归档候选人，建立企业人才库
- 🎯 **招聘渠道分析** - 分析各渠道招聘效果，优化招聘预算

## 适用场景

- 企业招聘流程自动化
- 简历筛选和人岗匹配
- 面试评估和决策
- 人才库建设管理

## 配置

在 `~/.openclaw/workspace/config/ai-hr-recruiter.json` 配置 SkillPay API Key。
