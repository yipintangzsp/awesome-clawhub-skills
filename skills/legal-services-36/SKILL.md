# 劳动争议处理与仲裁辅助系统

## 技能描述
劳动争议专业工具，劳动仲裁流程指导、赔偿计算、证据清单、文书生成、庭审准备，帮助劳动者和企业合法解决劳动纠纷。

## 定价
- **劳动者版**: ¥399/月
- **企业版**: ¥999/月
- **律所版**: ¥1999/月

## 核心功能
1. 劳动争议类型识别
2. 赔偿金自动计算
3. 证据清单生成
4. 仲裁文书生成
5. 庭审准备指导
6. 法律法规查询

## 使用方式
```bash
/labor-dispute activate
/labor-dispute calculate --type wrongful-termination --salary 15000 --years 3
/labor-dispute evidence --case wrongful-termination
/labor-dispute generate --document arbitration-application
```

## 适用场景
- 劳动者维权
- 企业 HR
- 劳动法律师
- 劳动仲裁委
