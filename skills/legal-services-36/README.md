# 劳动争议处理与仲裁辅助系统

## 产品概述

劳动争议专业处理平台，提供从咨询到仲裁到诉讼的全流程支持，帮助劳动者和企业依法解决劳动纠纷。

## 核心功能

### 争议识别
- 争议类型判断
- 法律关系分析
- 时效计算
- 管辖确定

### 赔偿计算
- 经济补偿金
- 赔偿金
- 工资差额
- 加班费
- 未休年假

### 证据管理
- 证据清单生成
- 证据收集指导
- 证据效力评估
- 举证期限提醒

### 文书生成
- 仲裁申请书
- 答辩状
- 证据目录
- 代理词

### 庭审准备
- 庭审流程
- 常见问题
- 辩论要点
- 注意事项
