# Brand Positioning AI - 品牌定位 AI

## 概述

利用 AI 技术进行品牌策略定位分析，找到差异化竞争优势。

## 服务内容包括

- 市场分析
- 竞品定位
- 目标人群画像
- 品牌价值主张
- 差异化策略

## 定价

- 单次：¥699/次
- 支持按行业复杂度定制价格

## 使用方法

```bash
brand-positioning-ai --industry <行业> --target <目标人群>
```

## 配置

在 `~/.openclaw/workspace/config/brand-positioning-ai.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_project": 699
}
```
