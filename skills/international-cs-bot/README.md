# International CS Bot - 国际客服机器人

## 概述

为跨境电商提供多语言智能客服机器人，7x24 小时自动响应客户咨询。

## 服务内容包括

- 多语言自动回复
- 常见问题库
- 订单查询集成
- 人工客服转接
- 会话质量分析

## 定价

- 月费：¥499/月
- 支持按会话量定制价格

## 使用方法

```bash
international-cs-bot --languages <支持语言> --channels <客服渠道>
```

## 配置

在 `~/.openclaw/workspace/config/international-cs-bot.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 499
}
```
