# 租房管理与收租自动化系统

## 技能描述
房东租房管理工具，租客管理、合同管理、自动收租、维修管理、账单管理、租期提醒，让收租更轻松。

## 定价
- **个人版**: ¥199/月（5 套）
- **专业版**: ¥499/月（20 套）
- **机构版**: ¥999/月（100 套）

## 核心功能
1. 租客信息管理
2. 租赁合同管理
3. 自动收租提醒
4. 维修工单管理
5. 水电账单管理
6. 租期到期提醒

## 使用方式
```bash
/rental-manager activate
/rental-manager add-tenant --info tenant.json
/rental-manager collect-rent --month 2024-03
/rental-manager maintenance --request REQ123
```

## 适用场景
- 个人房东
- 二房东
- 公寓运营
- 物业管理
