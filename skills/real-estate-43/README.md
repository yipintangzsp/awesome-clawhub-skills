# 租房管理与收租自动化系统

## 产品概述

租房管理平台，帮助房东高效管理出租房产，从租客管理到收租到维修，一站式解决租房管理问题。

## 核心功能

### 租客管理
- 租客信息
- 联系方式
- 入住记录
- 信用评估

### 合同管理
- 电子合同
- 租期管理
- 租金调整
- 续约提醒

### 收租管理
- 自动提醒
- 在线支付
- 逾期追踪
- 押金管理

### 维修管理
- 报修受理
- 工单派发
- 进度追踪
- 费用记录

### 账单管理
- 水电燃气
- 物业费
- 网络费
- 分摊计算
