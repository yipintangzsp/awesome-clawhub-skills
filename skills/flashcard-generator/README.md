## 分类
学习成长

## 价格
¥3/次

## 安装

```bash
clawhub install flashcard-generator
```

## 使用

```bash
flashcard-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/flashcard-generator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 3
}
```
