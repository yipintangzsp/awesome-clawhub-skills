# language-practice-ai

语言练习 AI（¥5/次）

## 分类
学习成长

## 价格
¥5/次

## 安装

```bash
clawhub install language-practice-ai
```

## 使用

```bash
language-practice-ai [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/language-practice-ai.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 5
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
