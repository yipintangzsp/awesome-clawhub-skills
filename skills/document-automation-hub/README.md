# Document Automation Hub - 文档自动化中心

## 概述

为企业提供智能文档自动化处理服务，从生成到审批全流程自动化。

## 服务内容包括

- 文档模板管理
- 自动数据填充
- 审批工作流
- 版本控制
- 电子签名

## 定价

- 月费：¥599/月
- 支持按文档量定制价格

## 使用方法

```bash
document-automation-hub --templates <文档模板> --workflow <工作流>
```

## 配置

在 `~/.openclaw/workspace/config/document-automation-hub.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
