- **天气预报** - 当天天气、温度、降水概率、穿衣建议

- **励志语录** - 每日一句正能量，开启好心情
- **定时推送** - 每天早上 8 点自动发送（可自定义时间）

## 📊 模块详解

### 🌤️ 天气模块
- 当天最高/最低温度
- 降水概率和降雨时段
- 空气质量指数（AQI）
- 紫外线强度
- 穿衣/出行建议

### 📆 日程模块
- 今日所有日历事件
- 会议时间和链接
- 待办事项提醒
- 截止日期预警

### 📧 邮件模块
- 未读邮件总数
- 重要邮件标题（最多 5 封）
- 发件人信息
- 邮件优先级标记

### 💰 收入模块
- 昨日收入及环比
- 本周累计及目标进度
- 本月累计及目标进度
- 爆款技能排行

### 💪 语录模块
- 每日一句正能量
- 名人名言标注
- 可切换类型（励志/幽默/哲学/商业）
