# 📅 Daily Briefing - 每日简报

**核心功能：**
- 天气预报 - 温度/降水概率/空气质量/穿衣建议
- 日程提醒 - 今日日历事件/会议时间/待办事项
- 邮件摘要 - 未读数量/重要邮件标题（最多 5 封）
- 收入数据 - SkillPay 昨日/本周/本月收入及目标进度
- 励志语录 - 每日一句正能量（可切换类型）
- 定时推送 - 每天早上 8 点自动发送（可自定义时间）

**定价：** ¥1/次 | ¥19/月 | ¥199/年

**典型场景：**
1. 晨间仪式 - 每天固定时间接收，形成稳定习惯
2. 日程管理 - 快速查看今天安排，避免忘记会议
3. 收入追踪 - 每天了解 SkillPay 收入，保持动力
4. 天气准备 - 根据天气决定穿衣和出行计划
5. 邮件处理 - 优先处理重要邮件，避免遗漏

**输入示例：** 
- `/daily-briefing setup` - 设置每日简报
- `/daily-briefing` - 手动获取今日简报
- `/daily-briefing time 07:30` - 修改推送时间
