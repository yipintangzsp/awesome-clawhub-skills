# 知乎标题优化 - 知乎高赞标题生成器

## 概述
基于知乎高赞回答和热门问题分析，生成高赞同、高曝光的标题。适配知乎社区调性和推荐算法。

## 定价
- **按次收费：** ¥5/次
- **包月订阅：** ¥99/月（无限次）

## 功能特性
- 🎯 知乎热榜分析
- 📊 赞同率预测
- 🔍 SEO 关键词优化
- 💬 问题式标题生成
- 📈 领域适配

## 使用方式
```bash
# 基础使用
/zhihu-title "内容主题"

# 指定领域
/zhihu-title "编程经验" --field 科技
/zhihu-title "职场困惑" --field 职场

# 问题优化
/zhihu-title --optimize "如何学习 Python？"

# 批量生成
/zhihu-title "理财知识" --count 5
```
