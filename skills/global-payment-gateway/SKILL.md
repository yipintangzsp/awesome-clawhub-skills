# Global Payment Gateway

全球支付网关 - 跨境支付解决方案

## 使用方式

```bash
global-payment-gateway --regions <目标区域> --methods <支付方式>
```

## 功能特点

- 💳 多支付方式
- 🌍 多货币支持
- 🔒 安全合规
- 📊 交易分析
- 💰 ¥699/月订阅

## 配置

在 `~/.openclaw/workspace/config/global-payment-gateway.json` 配置 SkillPay API Key。
