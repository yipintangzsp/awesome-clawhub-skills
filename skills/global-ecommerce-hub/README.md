# Global Ecommerce Hub - 全球电商中心

## 概述

为跨境电商提供一站式多平台管理解决方案，整合主流电商平台。

## 服务内容包括

- 多平台店铺管理
- 订单聚合处理
- 库存同步更新
- 多货币结算
- 销售数据分析

## 定价

- 月费：¥799/月
- 支持按店铺数量定制价格

## 使用方法

```bash
global-ecommerce-hub --platforms <平台列表> --markets <目标市场>
```

## 配置

在 `~/.openclaw/workspace/config/global-ecommerce-hub.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 799
}
```
