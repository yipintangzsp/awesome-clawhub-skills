# Global Ecommerce Hub

全球电商中心 - 跨境电商一站式管理

## 使用方式

```bash
global-ecommerce-hub --platforms <平台列表> --markets <目标市场>
```

## 功能特点

- 🌍 多平台管理
- 📦 订单聚合
- 💰 多货币结算
- 📊 数据分析
- 💰 ¥799/月订阅

## 配置

在 `~/.openclaw/workspace/config/global-ecommerce-hub.json` 配置 SkillPay API Key。
