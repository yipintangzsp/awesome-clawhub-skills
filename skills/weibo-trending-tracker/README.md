## 热度指标

```
热度值 = 讨论量 × 权重 1 + 阅读量 × 权重 2 + 视频播放 × 权重 3

讨论量：原创 + 转发 + 评论
阅读量：页面浏览次数
视频播放：相关视频播放量
```
