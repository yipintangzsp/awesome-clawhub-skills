# Weibo Trending Tracker - 微博热搜追踪

## Description
微博热搜监控工具，实时追踪热搜榜单、话题热度、讨论趋势。帮助品牌和个人把握热点，及时参与话题营销。

## Usage
```bash
# 查看热搜
/weibo-trending-tracker --action "list" --limit 20

# 话题分析
/weibo-trending-tracker --topic "话题名称" --action "analyze"

# 热度追踪
/weibo-trending-tracker --topic "话题名称" --action "track" --duration 24
```

### 参数说明
- `--topic`: 话题名称
- `--action`: 操作类型 (list, analyze, track, alert)
- `--limit`: 返回数量 (1-50)
- `--duration`: 追踪时长 (小时)
- `--category`: 热搜分类 (all, entertainment, social, news)
