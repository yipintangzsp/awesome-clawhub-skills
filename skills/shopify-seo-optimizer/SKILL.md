# Shopify SEO Optimizer - Shopify SEO 优化

## Description
专为 Shopify 店铺设计的 SEO 优化工具。优化产品标题、描述、元标签、图片 Alt、URL 结构，提升店铺在 Google 的搜索排名和自然流量。

## Usage
```bash
# 优化产品页面
/shopify-seo-optimizer --type "product" --handle "product-handle" --keyword "目标关键词"

# 优化集合页面
/shopify-seo-optimizer --type "collection" --handle "collection-handle"

# 批量优化
/shopify-seo-optimizer --action "bulk" --type "products" --limit 50

# SEO 审计
/shopify-seo-optimizer --action "audit" --url "your-store.com"
```

### 参数说明
- `--type`: 页面类型 (product, collection, page, blog, article)
- `--handle`: Shopify handle/URL
- `--keyword`: 目标关键词
- `--action`: 操作类型 (optimize, audit, suggest, bulk)
- `--limit`: 批量处理数量
