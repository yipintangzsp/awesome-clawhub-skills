# google-ads-optimizer

Google 广告优化（¥149/月）

## 分类
营销/广告优化

## 价格
¥149/月

## 安装

```bash
clawhub install google-ads-optimizer
```

## 使用

```bash
google-ads-optimizer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/google-ads-optimizer.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 149,
  "google_ads_api_key": "your_google_api_key",
  "target_roas": 400,
  "campaign_types": ["search", "display", "shopping"]
}
```

## 功能

- 广告账户全面诊断
- 关键词质量得分优化
- 智能出价策略调整
- 广告文案 A/B 测试
- 竞品广告监控分析
- 转化追踪优化
- ROAS 提升建议
