# google-ads-optimizer

Google 广告优化（¥149/月）

## 使用方式

```bash
google-ads-optimizer --help
```

## 功能特点

- 📊 广告绩效分析
- 🎯 关键词优化
- 💰 出价策略调整
- 📈 质量得分提升
- 🔍 竞品广告监控
- 🔒 SkillPay 收费集成

## 配置

在 `~/.openclaw/workspace/config/google-ads-optimizer.json` 配置 SkillPay API Key 和 Google Ads API。
