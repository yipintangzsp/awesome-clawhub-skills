# 投资者情绪分析与市场预警系统

## 技能描述
市场情绪监控工具，整合社交媒体、新闻、搜索数据、资金流向，AI 分析投资者情绪，识别市场极端状态，提供逆向投资信号。

## 核心功能
1. 社交媒体情绪分析
2. 新闻情感分析
3. 搜索趋势监控
4. 资金流向追踪
5. 极端情绪预警
6. 逆向信号生成

## 使用方式
```bash
/sentiment-analysis activate
/sentiment-analysis monitor --market A-share
/sentiment-analysis extreme --alert true
/sentiment-analysis contrarian --signal top-10
```
