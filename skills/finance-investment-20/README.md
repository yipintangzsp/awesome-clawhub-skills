# 投资者情绪分析与市场预警系统

## 产品概述

市场情绪分析平台，整合多维度情绪指标，AI 识别市场极端状态，为逆向投资和价值投资提供量化支持。

## 核心功能

### 情绪指标
- 社交媒体情绪（微博、雪球、股吧）
- 新闻情感分析
- 搜索指数（百度、谷歌）
- 论坛讨论热度

### 资金流向
- 北向资金
- 南向资金
- 融资融券
- 龙虎榜

### 极端预警
- 贪婪恐惧指数
- 超买超卖信号
- 拥挤度分析
- 黑天鹅预警

### 逆向信号
- 极度悲观买入信号
- 极度乐观卖出信号
- 分歧度分析
- 共识度追踪
