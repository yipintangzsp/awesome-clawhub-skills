# Strategic Planning AI - 战略规划 AI

## 概述

辅助企业制定战略规划，提供市场分析和策略建议。

## 服务内容包括

- 市场环境分析
- 竞争格局评估
- 战略目标制定
- 实施路径规划
- 风险评估

## 定价

- 单次：¥799/次
- 支持按规划周期定制价格

## 使用方法

```bash
strategic-planning-ai --industry <行业> --goals <战略目标>
```

## 配置

在 `~/.openclaw/workspace/config/strategic-planning-ai.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_plan": 799
}
```
