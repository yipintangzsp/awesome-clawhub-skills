# Etsy Tag Generator - Etsy 标签生成器

## Description
Etsy 店铺标签优化工具，基于产品特点和热门搜索词生成 13 个最佳标签。提升产品在 Etsy 搜索中的曝光率和点击率。

## Usage
```bash
# 生成标签
/etsy-tag-generator --title "Handmade Silver Ring" --category "jewelry"

# 优化已有标签
/etsy-tag-generator --action "optimize" --tags "ring,silver,handmade"

# 竞品标签分析
/etsy-tag-generator --action "competitor" --url "etsy.com/listing/xxx"
```

### 参数说明
- `--title`: 产品标题
- `--category`: 产品类别
- `--tags`: 已有标签（逗号分隔）
- `--action`: 操作类型 (generate, optimize, competitor, trend)
- `--marketplace`: 目标市场 (US, UK, EU, global)
