# gift-idea-generator

礼物创意生成（¥3/次）

## 分类
生活便利

## 价格
¥3/次

## 安装

```bash
clawhub install gift-idea-generator
```

## 使用

```bash
gift-idea-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/gift-idea-generator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 3
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
