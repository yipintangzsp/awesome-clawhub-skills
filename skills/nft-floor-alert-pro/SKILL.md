# nft-floor-alert-pro

NFT 地板价警报专业版（¥79/月）

## 使用方式

```bash
nft-floor-alert-pro --help
```

## 功能特点

- 📊 多平台地板价监控
- 🔔 实时价格警报
- 📈 7 天趋势分析
