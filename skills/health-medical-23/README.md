# AI 营养分析与饮食规划系统

## 产品概述

AI 驱动的营养管理平台，拍照识别食物，自动计算营养，根据目标生成个性化饮食方案，让健康饮食更简单。

## 核心功能

### 食物识别
- 拍照识别菜品
- 分量估算
- 热量和营养计算
- 外食/自制区分

### 营养分析
- 三大营养素追踪
- 维生素和矿物质
- 摄入量 vs 推荐量
- 营养评分

### 饮食规划
- 减肥食谱
- 增肌食谱
- 控糖食谱
- 孕期食谱

### 智能推荐
- 基于偏好推荐
- 基于库存推荐
- 外卖选择建议
- 购物清单生成
