# AI 营养分析与饮食规划系统

## 技能描述
智能营养管理工具，拍照识别食物、自动计算营养、个性化饮食规划、减肥/增肌/控糖方案、营养缺乏预警。

## 定价
- **标准版**: ¥299/月
- **专业版**: ¥599/月（+ 定制方案）
- **尊享版**: ¥999/月（+ 营养师咨询）

## 核心功能
1. 拍照识别食物和热量
2. 营养素自动计算
3. 个性化饮食规划
4. 减肥/增肌方案
5. 营养缺乏预警
6. 食谱推荐和购物清单

## 使用方式
```bash
/nutrition-ai activate
/nutrition-ai log --photo meal.jpg
/nutrition-ai plan --goal lose-weight --days 30
/nutrition-ai analyze --deficiency check
```

## 适用场景
- 减肥人群
- 健身人群
- 慢病饮食管理
- 孕期营养
