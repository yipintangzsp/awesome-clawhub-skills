# weather-outfit-advisor

天气穿搭建议（¥2/次）

## 分类
每日工具

## 价格
¥2/次

## 安装

```bash
clawhub install weather-outfit-advisor
```

## 使用

```bash
weather-outfit-advisor [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/weather-outfit-advisor.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 2
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
