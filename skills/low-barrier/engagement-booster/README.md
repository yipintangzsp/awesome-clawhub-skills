# engagement-booster

互动提升器（¥3/次）

## 分类
社交媒体

## 价格
¥3/次

## 安装

```bash
clawhub install engagement-booster
```

## 使用

```bash
engagement-booster [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/engagement-booster.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 3
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
