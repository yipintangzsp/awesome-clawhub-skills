## 分类
社交媒体

## 价格
¥2/次

## 安装

```bash
clawhub install comment-reply-ai
```

## 使用

```bash
comment-reply-ai [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/comment-reply-ai.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 2
}
```

- 自动收费验证
