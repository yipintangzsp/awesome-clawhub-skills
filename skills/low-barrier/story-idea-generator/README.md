# story-idea-generator

故事创意生成（¥2/次）

## 分类
社交媒体

## 价格
¥2/次

## 安装

```bash
clawhub install story-idea-generator
```

## 使用

```bash
story-idea-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/story-idea-generator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 2
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
