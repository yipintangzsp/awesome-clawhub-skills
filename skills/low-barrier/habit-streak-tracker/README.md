# habit-streak-tracker

习惯打卡追踪（¥1/次）

## 分类
每日工具

## 价格
¥1/次

## 安装

```bash
clawhub install habit-streak-tracker
```

## 使用

```bash
habit-streak-tracker [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/habit-streak-tracker.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 1
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
