## 使用方式

```bash
# 基础用法
calorie-counter-ai --food "米饭 200g 鸡胸肉 150g"

# 高级用法
calorie-counter-ai --photo food.jpg --goal lose_weight --bmr 1500
```

## 功能特点

- ⚡ **AI 食物识别** - 拍照自动识别食物，估算重量和热量
- 📊 **精准热量计算** - 10 万 + 食物数据库，中餐/西餐/零食全覆盖
- 🎯 **个性化建议** - 基于 BMI/目标/活动量，推荐每日热量摄入
- 📈 **饮食趋势分析** - 追踪饮食记录，找出热量超标原因
- 🍽️ **营养均衡分析** - 蛋白质/碳水/脂肪比例，科学饮食搭配
