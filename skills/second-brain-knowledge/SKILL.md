# knowledge-absorber - 知识吸收助手

## 描述
自动从文章、视频、播客、PDF 中提取知识点，生成结构化笔记，关联已有知识。支持多格式输入、智能摘要、知识卡片生成。让学习更高效。

## 定价
**¥149/月** - 知识管理订阅

## 功能特性
- 📄 多格式内容解析
- 🧠 智能知识点提取
- 📝 结构化笔记生成
- 🔗 知识关联推荐
- 📇 知识卡片创建
- 📚 学习进度追踪

## 用法
```bash
# 安装技能
openclaw skills install knowledge-absorber

# 吸收文章
openclaw absorb article <URL>

# 吸收视频
openclaw absorb video <URL>

# 吸收 PDF
openclaw absorb pdf <文件路径>

# 生成知识卡片
openclaw absorb card <主题>

# 查看知识库
openclaw absorb library
```
