# powerpoint-designer

PPT 设计师（SkillPay 收费版）

## 使用方式

```bash
powerpoint-designer --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/powerpoint-designer.json` 配置 SkillPay API Key。
