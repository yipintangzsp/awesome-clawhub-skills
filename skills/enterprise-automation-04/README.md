# 员工考勤与薪资自动化系统

## 产品概述

一体化考勤薪资管理系统，实现从考勤打卡到工资发放的全流程自动化，降低 HR 工作量 80% 以上。

## 核心功能

### 考勤管理
- 多种打卡方式
- 自动识别异常考勤
- 请假加班自动关联
- 考勤报表自动生成

### 智能排班
- AI 优化排班
- 自动处理调班
- 合规性检查
- 成本优化建议

### 薪资核算
- 自动计算应发工资
- 个税专项扣除
- 社保公积金计算
- 工资条自动发送
