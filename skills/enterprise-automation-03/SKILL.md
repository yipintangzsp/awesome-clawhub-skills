# 发票与报销自动处理系统

## 技能描述
企业级发票 OCR 识别、自动分类、报销流程自动化系统。支持多格式发票识别、自动匹配订单、智能审核、对接财务系统，大幅降低财务人工成本。

## 定价
- **标准版**: ¥799/月（500 张/月）
- **专业版**: ¥1599/月（2000 张/月）
- **企业版**: ¥2999/月（无限）

## 核心功能
1. 多格式发票 OCR 识别（纸质/电子/截图）
2. 自动分类和科目匹配
3. 智能审核和异常检测
4. 报销流程自动化
5. 财务系统对接（金蝶/用友/SAP）
6. 税务合规检查

## 使用方式
```bash
/invoice-automation activate
/invoice-automation scan --file invoice.pdf
/invoice-automation submit --batch 2024-03
/invoice-automation report --month 2024-03
```

## 适用场景
- 企业财务部门发票处理
- 员工报销自动化
- 跨境电商多币种发票
- 连锁门店发票集中管理
