# 发票与报销自动处理系统

## 产品概述

智能发票处理系统，通过 OCR 和 AI 技术实现发票自动识别、分类、审核和入账，将财务人员从繁琐的手工录入中解放出来。

## 核心功能

### 发票识别
- 支持增值税发票、电子发票、收据等
- OCR 识别准确率 99%+
- 支持多语言和多币种
- 自动去重和验真

### 智能审核
- 自动匹配采购订单
- 异常金额预警
- 重复报销检测
- 合规性检查

### 流程自动化
- 自动创建报销单
- 智能路由审批
- 自动入账
- 付款提醒

## 集成系统
- 金蝶云
- 用友 NC
- SAP
- Oracle Financials
- 钉钉审批
- 企业微信
