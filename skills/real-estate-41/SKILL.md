# 房地产投资分析与 ROI 计算系统

## 技能描述
房地产投资分析工具，房源数据分析、租金回报率计算、现金流预测、增值潜力评估、投资风险识别，帮助投资者做出明智决策。

## 核心功能
1. 房源数据采集和分析
2. 租金回报率计算
3. 现金流预测模型
4. 增值潜力评估
5. 投资风险识别
6. 投资对比分析

## 使用方式
```bash
/rei-analysis activate
/rei-analysis analyze --property PROP123
/rei-analysis roi --price 3000000 --rent 12000
/rei-analysis compare --properties [PROP1,PROP2,PROP3]
```
