# Innovation Pipeline - 创新管道

## 概述

管理企业创新项目从创意到落地的全流程，提升创新成功率。

## 服务内容包括

- 创意收集管理
- 项目评估筛选
- 孵化流程管理
- 资源协调分配
- 进展追踪报告

## 定价

- 月费：¥799/月
- 支持按项目数量定制价格

## 使用方法

```bash
innovation-pipeline --ideas <创意来源> --stage <创新阶段>
```

## 配置

在 `~/.openclaw/workspace/config/innovation-pipeline.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 799
}
```
