## 使用方式

```bash
# 基础用法
ai-data-pipeline --source <数据源> --target <目标>

# 高级用法
ai-data-pipeline --source mysql,api --target warehouse --transform clean --schedule daily
```

## 功能特点

- ⚡ **AI 自动管道构建** - 自然语言描述需求，AI 自动生成 ETL 代码
- 🧹 **智能数据清洗** - 自动检测并修复缺失值、异常值、格式问题
- 🔄 **多源数据集成** - 数据库/API/文件/流数据，一站式整合
- 📊 **实时监控大屏** - 管道运行状态、数据量、延迟可视化
- 🔔 **异常自动告警** - 故障自动检测，短信/邮件/钉钉多通道告警

## 适用场景

- 📊 **数据仓库** - 多源数据自动同步，构建统一数据仓库
- 📈 **BI 报表** - 定时更新数据，支撑实时业务分析
- 🤖 **AI 训练** - 自动化特征工程，为 ML 模型准备高质量数据
- 🌐 **数据中台** - 企业级数据集成，打通数据孤岛
