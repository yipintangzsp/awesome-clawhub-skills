# staking-reward-tracker

质押收益追踪（SkillPay 收费版）

## 分类
Web3/加密货币

## 价格
¥29/月

## 安装

```bash
clawhub install staking-reward-tracker
```

## 使用

```bash
staking-reward-tracker [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/staking-reward-tracker.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
