# staking-reward-tracker

质押收益追踪（SkillPay 收费版）

## 使用方式

```bash
staking-reward-tracker --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/staking-reward-tracker.json` 配置 SkillPay API Key。
