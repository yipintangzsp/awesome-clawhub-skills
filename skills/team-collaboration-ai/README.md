# Team Collaboration AI - 团队协作 AI

## 概述

优化团队协作效率，智能协调任务和沟通，提升团队生产力。

## 服务内容包括

- 智能任务分配
- 沟通优化
- 进度追踪
- 冲突检测
- 团队分析

## 定价

- 月费：¥499/月
- 支持按团队规模定制价格

## 使用方法

```bash
team-collaboration-ai --team <团队信息> --tools <协作工具>
```

## 配置

在 `~/.openclaw/workspace/config/team-collaboration-ai.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 499
}
```
