# screenshot-ocr-pro

截图 OCR 专业版（SkillPay 收费版）

## 使用方式

```bash

```

## 功能特点

- 🔒 安全可靠

在 `~/.openclaw/workspace/config/screenshot-ocr-pro.json` 配置 SkillPay API Key。
