# REITs 投资分析与配置系统

## 技能描述
REITs 投资工具，公募 REITs 数据分析、估值分析、分红预测、组合配置、风险提示，帮助投资者参与不动产证券化投资。

## 定价
- **标准版**: ¥399/月
- **专业版**: ¥899/月
- **机构版**: ¥1999/月

## 核心功能
1. REITs 数据采集
2. 估值分析
3. 分红预测
4. 组合配置建议
5. 风险提示
6. 市场对比

## 使用方式
```bash
/reits-invest activate
/reits-invest analyze --reits REIT123
/reits-invest dividend-forecast --reits REIT456
/reits-invest portfolio --risk moderate --capital 1000000
```

## 适用场景
- REITs 投资者
- 资产配置
- 被动收入
- 机构投资者
