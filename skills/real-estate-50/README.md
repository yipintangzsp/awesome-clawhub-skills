# REITs 投资分析与配置系统

## 产品概述

REITs 投资分析平台，提供公募 REITs 的全面数据分析、估值分析、分红预测，帮助投资者参与不动产证券化投资。

## 核心功能

### 数据分析
- 基础信息
- 底层资产
- 运营数据
- 财务数据

### 估值分析
- NAV 计算
- 相对估值
- DCF 估值
- 估值区间

### 分红预测
- 历史分红
- 分红率分析
- 未来预测
- 税收影响

### 组合配置
- 风险测评
- 配置建议
- 分散策略
- 再平衡

### 风险提示
- 利率风险
- 市场风险
- 运营风险
- 流动性风险
