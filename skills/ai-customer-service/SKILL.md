# ai-customer-service - AI 智能客服自动化系统

**痛点**：客服咨询量大，重复问题多，响应慢，人力成本高，客户满意度低
**价值**：AI 自动处理 80% 常见问题，24/7 在线响应，降低客服成本 60%+

## 使用方式

```bash
ai-customer-service --help
```

## 功能特点

- ⚡ **智能自动问答** - AI 理解问题意图，自动给出准确回答
- 📊 **工单智能分类** - 自动识别问题类型，分派给对应部门
- 💰 **多轮对话支持** - 支持复杂问题多轮对话，提升解决率
- 🔒 **情绪识别预警** - 识别客户情绪，升级处理潜在投诉
- 🎯 **知识库自学习** - 从历史对话自动学习，持续优化回答质量

## 适用场景

- 电商客服自动化
- 企业客服系统
- 在线咨询服务
- 客户支持工单管理

## 配置

在 `~/.openclaw/workspace/config/ai-customer-service.json` 配置 SkillPay API Key。
