# 老年人健康监护与紧急呼叫系统

## 产品概述

老年人远程健康监护平台，整合智能设备数据，提供健康监测、紧急呼叫、日常关怀等服务，让老人安全、子女安心。

## 核心功能

### 健康监测
- 血压、血糖、心率
- 血氧、体温
- 体重、活动量
- 睡眠质量

### 用药管理
- 定时提醒
- 服药确认
- 漏服通知
- 药品库存

### 安全监护
- 跌倒检测
- 异常行为识别
- 紧急呼叫
- 位置追踪

### 家属联动
- 实时数据查看
- 异常通知
- 日报/周报
- 远程关怀

## 支持设备
- 智能血压计
- 血糖仪
- 智能手环
- 跌倒检测器
- 紧急按钮
