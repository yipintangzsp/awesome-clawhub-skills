# 老年人健康监护与紧急呼叫系统

## 技能描述
老年人健康管理平台，远程健康监测、用药提醒、跌倒检测、紧急呼叫、家属通知，让子女安心、老人安全。

## 定价
- **标准版**: ¥499/月
- **专业版**: ¥899/月（+ 设备）
- **尊享版**: ¥1499/月（+24h 监护）

## 核心功能
1. 健康指标远程监测
2. 用药提醒和确认
3. 跌倒检测和报警
4. 一键紧急呼叫
5. 家属实时通知
6. 日常活动追踪

## 使用方式
```bash
/elder-care activate
/elder-care monitor --devices bp-monitor,glucose-meter
/elder-care emergency --contact daughter,son
/elder-care daily-report --send family-group
```

## 适用场景
- 独居老人
- 慢病老人
- 失能老人
- 养老机构
