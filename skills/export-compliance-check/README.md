# Export Compliance Check - 出口合规检查

## 概述

为跨境电商提供出口合规审核服务，确保产品符合目标国家法规要求。

## 服务内容包括

- 出口许可检查
- 产品合规审核
- 禁运清单筛查
- 认证要求指导
- 风险评估报告

## 定价

- 月费：¥499/月
- 支持按审核次数定制价格

## 使用方法

```bash
export-compliance-check --product <产品信息> --destination <目的地>
```

## 配置

在 `~/.openclaw/workspace/config/export-compliance-check.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 499
}
```
