## 使用方式

```bash
# 基础用法
ai-data-analyst --data path/to/data.csv

# 高级用法
ai-data-analyst --data sales.csv --type trend --output pdf --language zh
```

## 功能特点

- ⚡ **智能数据诊断** - 自动检测数据质量、异常值、缺失值，给出清洗建议
- 📊 **趋势分析预测** - 识别销售/用户/流量趋势，预测未来走势
- 💡 **业务洞察提炼** - 用通俗语言解释数据背后的业务含义和机会
- 📈 **可视化报告生成** - 自动生成图表 + 文字报告，支持 PPT/PDF 导出
- 🔍 **多维对比分析** - 同比/环比/竞品对比，快速定位问题和机会

## 适用场景

- 📈 **销售分析** - 业绩追踪、区域对比、产品分析、预测下月目标
- 👥 **用户分析** - 用户增长、留存率、行为分析、分群画像
- 💰 **财务分析** - 收支分析、成本结构、利润率、现金流预测
- 📊 **运营分析** - 流量分析、转化漏斗、渠道效果、ROI 评估
