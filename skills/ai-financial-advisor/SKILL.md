# ai-financial-advisor - AI 智能财务顾问系统

**痛点**：理财知识缺乏，投资决策盲目，资产配置不合理，财富增值慢
**价值**：AI 专业财务规划，科学资产配置，实现财富稳健增值

## 使用方式

```bash
ai-financial-advisor --help
```

## 功能特点

- ⚡ **财务健康诊断** - 全面分析收支、资产负债、保险等财务状况
- 📊 **投资组合分析** - 评估现有投资，识别风险和优化空间
- 💰 **理财规划建议** - 基于目标生成个性化理财方案
- 🔒 **风险评估管理** - 评估风险承受能力，匹配合适投资产品
- 🎯 **财富增值策略** - 基于市场数据推荐最优投资策略

## 适用场景

- 个人财务规划
- 投资组合优化
- 退休养老规划
- 财富管理咨询

## 配置

在 `~/.openclaw/workspace/config/ai-financial-advisor.json` 配置 SkillPay API Key。
