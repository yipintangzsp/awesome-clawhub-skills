# ai-financial-advisor

AI 财务顾问（¥299/月）

## 分类
AI/财务

## 价格
¥299/月

## 安装

```bash
clawhub install ai-financial-advisor
```

## 使用

```bash
ai-financial-advisor [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/ai-financial-advisor.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 29,
  "risk_profile": "moderate",
  "investment_goals": ["retirement", "wealth_growth"],
  "time_horizon": "10y"
}
```

## 功能

- 财务健康评估
- 投资组合建议
- 退休规划
- 税务优化建议
- SkillPay 集成
