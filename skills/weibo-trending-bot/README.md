# 微博热搜机器人 🔥

实时监控微博热搜，自动追踪热点，生成蹭热点文案，提升账号曝光。

## 功能特性

- 🔥 **热搜榜单**: 实时微博热搜、文娱榜、社会榜
- 📊 **趋势分析**: 热度变化曲线
- ✍️ **文案生成**: AI 自动生成蹭热点文案
- 🔔 **智能提醒**: 关键词监控推送
- 📈 **数据追踪**: 话题热度预测

## 使用示例

### 查看热搜榜
```
/weibo-trending-bot --hot
```

### 按分类查看
```
/weibo-trending-bot --hot --category 文娱
```

### 生成蹭热点文案
```
/weibo-trending-bot --generate --topic "#AI 技术突破#"
```

### 监控关键词
```
/weibo-trending-bot --alert --keyword "科技创新"
```
