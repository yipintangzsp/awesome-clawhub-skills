# 婚姻家事法律服务平台

## 技能描述
婚姻家事法律工具，离婚流程指导、财产分割计算、抚养权分析、文书生成、调解辅助，专业处理婚姻家事法律问题。

## 定价
- **标准版**: ¥499/月
- **专业版**: ¥999/月
- **尊享版**: ¥1999/月（+ 律师咨询）

## 核心功能
1. 离婚流程指导
2. 财产分割计算
3. 抚养权分析
4. 抚养费计算
5. 文书模板生成
6. 调解方案建议

## 使用方式
```bash
/family-law activate
/family-law divorce-guide --type contested
/family-law property-split --assets assets.json --debts debts.json
/family-law custody --children 2 --income-parents [20000,15000]
```

## 适用场景
- 离婚咨询
- 财产分割
- 抚养权纠纷
- 遗产继承
