# 婚姻家事法律服务平台

## 产品概述

婚姻家事法律专业平台，提供离婚、财产分割、子女抚养、遗产继承等家事法律服务，帮助用户妥善处理家庭法律事务。

## 核心功能

### 离婚服务
- 协议/诉讼离婚指导
- 离婚流程说明
- 材料清单
- 时间预估

### 财产分割
- 共同财产识别
- 财产价值评估
- 分割方案建议
- 隐匿财产调查

### 子女抚养
- 抚养权分析
- 抚养费计算
- 探望权安排
- 变更抚养

### 遗产继承
- 继承人确定
- 遗产范围
- 遗嘱效力
- 继承流程

### 文书服务
- 离婚协议
- 财产协议
- 遗嘱
- 起诉状
