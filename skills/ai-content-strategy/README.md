## 安装

```bash
clawhub install ai-content-strategy
```

## 使用

```bash
ai-content-strategy [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/ai-content-strategy.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 19,
  "platforms": ["Blog", "Social Media", "Email"],
  "content_types": ["文章", "视频脚本", "图文"]
}
```

## 功能

- 内容策略规划
- 内容日历生成
- 选题建议
- 竞品内容分析
- SkillPay 集成
