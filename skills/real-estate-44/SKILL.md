# 房贷计算与优化系统

## 技能描述
房贷计算工具，贷款方案对比、月供计算、提前还款分析、转贷优化、利率追踪，帮助购房者选择最优贷款方案。

## 定价
- **标准版**: ¥199/月
- **专业版**: ¥499/月
- **尊享版**: ¥899/月（+ 银行对接）

## 核心功能
1. 房贷计算器
2. 贷款方案对比
3. 提前还款分析
4. 转贷优化建议
5. 利率追踪提醒
6. 公积金组合贷

## 使用方式
```bash
/mortgage-calc activate
/mortgage-calc calculate --price 5000000 --down 1500000 --years 30
/mortgage-calc compare --plans [plan1,plan2,plan3]
/mortgage-calc prepay --remaining 2000000 --extra 500000
```

## 适用场景
- 购房贷款
- 房贷优化
- 提前还款
- 转贷决策
