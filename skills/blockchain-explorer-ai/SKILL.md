# blockchain-explorer-ai

区块链浏览器 AI（SkillPay 收费版）

## 使用方式

```bash
blockchain-explorer-ai --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/blockchain-explorer-ai.json` 配置 SkillPay API Key。
