# layer2-bridge-finder

Layer2 桥接 finder（SkillPay 收费版）

## 使用方式

```bash
layer2-bridge-finder --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/layer2-bridge-finder.json` 配置 SkillPay API Key。
