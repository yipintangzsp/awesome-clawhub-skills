# 广告文案 A/B 测试 - 广告效果优化

## 概述
生成多版本广告文案，提供 A/B 测试方案，优化广告点击率和转化率。支持多平台：抖音、Facebook、Google、微信广告。

## 定价
- **按次收费：** ¥9/次
- **包月订阅：** ¥199/月（无限次）

## 功能特性
- 📝 多版本文案生成
- 📊 CTR/CVR 预测
- 🎯 平台适配优化
- 🧪 A/B 测试方案
- 📈 效果追踪建议

## 使用方式
```bash
# 基础生成
/ad-copy "产品/服务描述" --platform 抖音

# 多版本
/ad-copy "产品" --platform Facebook --count 5

# A/B 测试
/ad-copy "产品" --ab-test

# 竞品分析
/ad-copy "产品" --analyze "竞品广告"
```

## 输出格式
```markdown
## 📝 广告文案 A/B 测试方案

### 🎯 产品信息
- 产品：XXX
- 平台：抖音
- 目标：转化

### 📊 文案版本

#### 版本 A（情感诉求）
**标题：** "30 天，她瘦了 20 斤"
**正文：** "不节食不运动，这个方法太狠了..."
**CTA：** "立即免费试用"

#### 版本 B（利益诉求）

### 🏆 推荐版本
**版本 B** - 利益诉求更适合抖音平台

### 🧪 A/B 测试方案
- 预算分配：50% / 50%
- 测试周期：7 天
- 成功指标：CTR 提升 20%
```
