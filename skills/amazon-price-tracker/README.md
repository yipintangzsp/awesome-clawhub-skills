# Amazon Price Tracker 📊

实时监控亚马逊商品价格，历史价格分析，降价自动提醒，助你低价购入好物。

## 功能特性

- 📉 **价格追踪**: 7x24 小时监控价格变化
- 📈 **历史曲线**: 查看 90 天价格历史
- 🔔 **降价提醒**: 邮件/消息通知
- 🎯 **目标价格**: 设置心理价位自动提醒
- 📊 **竞品分析**: 卖家竞品价格监控

## 使用示例

### 添加追踪商品
```
/amazon-price-tracker --add https://amazon.com/dp/B08N5WRWNW
  --target-price 99
  --notify email
```

### 查看追踪列表
```
/amazon-price-tracker --list
```

### 查看价格历史
```
/amazon-price-tracker --history https://amazon.com/dp/B08N5WRWNW
```

### 设置提醒方式
```
/amazon-price-tracker --alert --email user@example.com
/amazon-price-tracker --alert --telegram @username
```
