## 安装

```bash
clawhub install grammar-checker-lite
```

## 使用

```bash
grammar-checker-lite [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/grammar-checker-lite.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 5
}
```
