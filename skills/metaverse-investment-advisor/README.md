# metaverse-investment-advisor

元宇宙投资顾问（¥199/月）

## 分类
Web3/元宇宙

## 价格
¥199/月

## 安装

```bash
clawhub install metaverse-investment-advisor
```

## 使用

```bash
metaverse-investment-advisor [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/metaverse-investment-advisor.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 19,
  "focus_areas": ["虚拟地产", "GameFi", "社交元宇宙"],
  "risk_tolerance": "medium"
}
```

## 功能

- 元宇宙项目评估
- 投资价值分析
- 风险等级评定
- 市场趋势预测
- SkillPay 集成
