# shopify-conversion-optimizer-pro - Shopify 转化优化专业版

## 技能描述
专业版 Shopify 店铺转化优化工具，深度分析用户行为，提供页面优化、结账流程、产品页改进建议。支持热力图分析、A/B 测试、弃单挽回。

## 功能特性
- 📊 转化漏斗深度分析
- 🛒 购物车弃单分析
- 🎨 页面热力图分析
- 📱 移动端体验优化
- 💳 结账流程优化
- 📧 弃单邮件自动化

## 使用方式
```bash
# 分析转化率
/shopify-conversion-optimizer-pro --analyze --store <店铺域名>

# 弃单分析
/shopify-conversion-optimizer-pro --abandoned --days 30

# 页面优化建议
/shopify-conversion-optimizer-pro --page <产品页 URL>

# A/B 测试建议
/shopify-conversion-optimizer-pro --abtest --goal conversion
```
