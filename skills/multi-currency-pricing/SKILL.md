# Multi Currency Pricing

多货币定价 - 智能动态定价系统

## 使用方式

```bash
multi-currency-pricing --base-currency <基础货币> --target-currencies <目标货币>
```

## 功能特点

- 💱 实时汇率
- 📊 动态定价
- 💰 利润保护
- 🌍 区域定价
- 💰 ¥399/月订阅

## 配置

在 `~/.openclaw/workspace/config/multi-currency-pricing.json` 配置 SkillPay API Key。
