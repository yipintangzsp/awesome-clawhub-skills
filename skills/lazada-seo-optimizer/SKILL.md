# Lazada SEO Optimizer

Lazada SEO 优化工具 - 优化产品标题和描述提升搜索排名

## 使用方式

```bash
lazada-seo-optimizer --product "产品标题" --keywords "关键词 1，关键词 2"
```

## 功能特点

- 🔑 关键词优化
- 📝 标题生成
- 📊 排名预测
- 💰 ¥29/月订阅

## 配置

在 `~/.openclaw/workspace/config/lazada-seo-optimizer.json` 配置 SkillPay API Key。
