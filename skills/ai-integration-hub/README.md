# AI Integration Hub - AI 集成中心

## 概述

提供一站式 AI 服务集成平台，简化多 AI 服务的接入和管理。

## 服务内容包括

- 多模型 API 统一接入
- 服务编排与工作流
- 统一认证与鉴权
- 请求路由与负载均衡
- 监控与日志聚合

## 定价

- 月费：¥699/月
- 支持按集成服务数定制价格

## 使用方法

```bash
ai-integration-hub --services <服务列表> --platform <目标平台>
```

## 配置

在 `~/.openclaw/workspace/config/ai-integration-hub.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 699
}
```
