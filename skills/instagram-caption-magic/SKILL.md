# Instagram Caption Magic - Instagram 文案魔法

## Description
Instagram 文案生成工具，基于图片内容和品牌风格生成高互动文案。包含 emoji、标签、CTA，提升点赞、评论和粉丝增长。

## Usage
```bash
# 生成文案
/instagram-caption-magic --type "photo" --vibe "aesthetic" --topic "travel"

# 生成标签
/instagram-caption-magic --action "hashtags" --niche "fashion" --count 30

# 竞品分析
/instagram-caption-magic --action "analyze" --username "@brand"
```

### 参数说明
- `--type`: 内容类型 (photo, reel, carousel, story)
- `--vibe`: 风格氛围 (aesthetic, funny, inspirational, casual, luxury)
- `--topic`: 内容主题
- `--niche`: 领域 (fashion, food, travel, fitness, beauty, business)
- `--action`: 操作类型 (caption, hashtags, analyze, optimize)
