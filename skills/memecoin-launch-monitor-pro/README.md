# Meme Coin Launch Monitor Pro

专业版土狗币监控工具，帮助你在新币发射中抓住机会、避开陷阱。

## 核心功能

### 1. 新币发射监控
- Pump.fun 新币实时追踪
- DexScreener 热门新币
- Twitter/Telegram 热度监控
- 早期价格异常检测

### 2. 安全审计
- 合约代码扫描
- 后门/honeypot 检测
- 铸币权限检查
- 交易税分析

### 3. 流动性分析
- 流动性池大小
- LP 锁定状态和时长
- 流动性提供者分布
- 撤池风险评估

### 4. 持币分析
- 持币者数量增长
- Top10 持币占比
- 内部人士持仓检测
- 巨鲸钱包追踪

### 5. 风险评估
- 综合风险评分（0-10）
-  Rug Pull 概率
-  流动性风险
- 合约风险
