# 微博热搜爬虫

## 描述
自动爬取微博实时热搜榜单，生成包含热搜排名、话题、热度值的 Markdown 报告。

## 触发条件
当用户说"爬微博热搜"、"微博热搜"、"查看微博热门"时使用此技能。
当用户只是闲聊或询问与微博无关的问题时，不使用此技能。

## 指令
按以下步骤执行：
1. 使用 curl 访问微博热搜 API 或网页
2. 解析热搜榜单数据（排名、话题、热度值）
3. 生成 Markdown 格式报告
4. 保存到 data/weibo/ 目录

## 输出格式
必须严格按照以下格式输出：

```markdown
# 微博实时热搜榜 - YYYY-MM-DD HH:mm

| 排名 | 话题 | 热度值 |
|------|------|--------|
| 1 | xxx | xxx |
| 2 | xxx | xxx |
| ... | ... | ... |
```

## 约束条件
- 不要修改原始网页内容
- 不要生成超过 50 条热搜
- 如果无法访问，返回错误信息而不是猜测内容
