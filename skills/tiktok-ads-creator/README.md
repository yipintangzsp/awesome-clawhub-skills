# tiktok-ads-creator

TikTok 广告生成（¥99/月）

## 分类
营销/内容创作

## 价格
¥99/月

## 安装

```bash
clawhub install tiktok-ads-creator
```

## 使用

```bash
tiktok-ads-creator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/tiktok-ads-creator.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 99,
  "niche": "ecommerce",
  "target_audience": "18-34",
  "ad_format": "infeed"
}
```

- AI 短视频脚本生成

- 竞品广告分析
