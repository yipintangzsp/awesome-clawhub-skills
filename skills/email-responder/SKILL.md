# email-responder

邮件自动回复（SkillPay 收费版）

## 使用方式

```bash
email-responder --help
```

## 功能特点

- ⚡ 快速执行
- 💰 SkillPay 收费集成
- 🔒 安全可靠

## 配置

在 `~/.openclaw/workspace/config/email-responder.json` 配置 SkillPay API Key。
