# crypto-tax-optimizer

加密货币税务优化（¥129/月）

## 使用方式

```bash
crypto-tax-optimizer --help
```

## 功能特点

- 📊 多交易所交易导入
- 🧮 自动盈亏计算
- 📑 税务报告生成
- 💡 税务优化策略
- 🌍 多国税务规则支持
- 🔒 SkillPay 收费集成

## 配置

在 `~/.openclaw/workspace/config/crypto-tax-optimizer.json` 配置 SkillPay API Key 和交易所 API。
