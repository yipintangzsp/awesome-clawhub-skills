# global-logistics-tracker - 全球物流智能追踪系统

**痛点**：物流信息分散，无法统一追踪，运费成本高，时效难预测
**价值**：聚合 100+ 承运商数据，智能对比运费，降低物流成本 20-30%

## 使用方式

```bash
global-logistics-tracker --help
```

## 功能特点

- ⚡ **多承运商聚合** - 支持 DHL/FedEx/UPS/邮政等 100+ 物流商统一追踪
- 📊 **运费智能对比** - 实时对比各承运商价格，推荐最优方案
- 💰 **时效预测分析** - 基于历史数据预测送达时间，管理客户预期
- 🔒 **异常物流预警** - 延误、丢件、清关问题实时提醒
- 🎯 **物流成本分析** - 多维度分析物流费用，找出优化空间

## 适用场景

- 跨境电商订单追踪
- 物流成本优化
- 供应链时效管理
- 客户物流查询服务

## 配置

在 `~/.openclaw/workspace/config/global-logistics-tracker.json` 配置 SkillPay API Key。
