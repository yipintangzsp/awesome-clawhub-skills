# 小红书自动发布 📕

自动化小红书内容发布工具，智能标签优化 + 定时发布，提升笔记曝光。

## 功能特性

- 📝 **自动发布**: 支持图文笔记自动发布
- ⏰ **定时发布**: 设置最佳发布时间
- 🏷️ **标签优化**: AI 生成热门标签
- 🖼️ **封面建议**: 分析热门封面风格
- 📊 **数据分析**: 追踪笔记表现

## 使用示例

### 发布图文笔记
```
/xiaohongshu-auto-post --content "今天分享我的护肤心得..." 
  --images image1.jpg,image2.jpg,image3.jpg
  --title "28 天护肤挑战，皮肤真的变好了！"
```

### 定时发布
```
/xiaohongshu-auto-post --content "内容..." 
  --schedule "2026-03-10 18:30"
```

### 分析热门内容
```
/xiaohongshu-auto-post --analyze --category 护肤 --days 7
```

### 生成标签
```
/xiaohongshu-auto-post --hashtags "护肤 美白 抗老"
```
