# xiaohongshu-auto-post - 小红书自动发布

## 描述
自动发布小红书笔记，支持定时发布、标签优化、封面生成。分析热门内容，提供发布建议。

## 定价
- **包月订阅**: ¥19/月
- 每日最多发布 5 篇笔记
- 包含标签优化和封面建议

## 用法
```bash
# 发布笔记
/xiaohongshu-auto-post --content "笔记内容" --images img1.jpg,img2.jpg

# 定时发布
/xiaohongshu-auto-post --content "内容" --schedule "2026-03-10 18:00"

# 分析热门
/xiaohongshu-auto-post --analyze --category 美妆

# 生成标签
/xiaohongshu-auto-post --hashtags "护肤心得"
```
