# tokenomics-analyzer - 代币经济模型深度分析工具

**痛点**：代币经济复杂难懂，释放计划不透明，盲目投资导致被割韭菜
**价值**：专业代币经济分析，科学估值建模，避开 90% 的垃圾项目

## 使用方式

```bash
tokenomics-analyzer --help
```

## 功能特点

- ⚡ **代币模型分析** - 深度解析供应总量、分配机制、通胀模型
- 📊 **释放计划追踪** - 可视化代币解锁时间表，预警大额解锁抛压
- 💰 **估值建模** - 基于 FDV/TVL/收入等多维度科学估值
- 🔒 **风险预警** - 识别团队占比过高、解锁过于集中等风险
- 🎯 **投资建议** - 基于代币经济给出买入时机和仓位建议

## 适用场景

- 新币投资前尽职调查
- 持仓代币风险评估
- 空投项目价值分析
- 加密投资组合优化

## 配置

在 `~/.openclaw/workspace/config/tokenomics-analyzer.json` 配置 SkillPay API Key。
