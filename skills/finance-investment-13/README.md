# 量化交易策略回测与自动执行系统

## 产品概述

专业级量化交易平台，从策略研发到实盘执行一站式解决方案，支持多市场、多策略、多账户管理。

## 核心功能

### 策略开发
- Python/可视化双模式
- 内置 100+ 策略模板
- 自定义指标库
- 版本管理

### 回测引擎
- 高精度历史数据
- 考虑手续费和滑点
- 多周期并行回测
- 详细绩效报告

### 参数优化
- 网格搜索
- 遗传算法
- 贝叶斯优化
- 过拟合检测

### 实盘执行
- 低延迟订单执行
- 智能订单路由
- 异常处理
- 日志和监控

### 风险管理
- 仓位限制
- 止损止盈
- 回撤控制
- 风险指标监控

## 支持市场
- A 股、港股、美股
- 商品期货、股指期货
- 加密货币现货/合约
- 外汇
