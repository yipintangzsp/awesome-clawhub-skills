# 量化交易策略回测与自动执行系统

## 技能描述
专业量化交易平台，策略编写、历史回测、参数优化、模拟交易、实盘自动执行、风险控制，支持股票、期货、加密货币多市场。

## 定价
- **标准版**: ¥1299/月（股票/期货）
- **专业版**: ¥2299/月（+ 加密货币）
- **机构版**: ¥2999/月（+ 低延迟/定制）

## 核心功能
1. 可视化策略编辑器
2. 多周期历史回测
3. 参数优化和过拟合检测
4. 模拟交易验证
5. 实盘自动执行
6. 风险控制和资金管理

## 使用方式
```bash
/quant-trading activate
/quant-trading backtest --strategy ma_cross --period 5y
/quant-trading optimize --params window,threshold
/quant-trading deploy --strategy ma_cross --capital 100000
```
