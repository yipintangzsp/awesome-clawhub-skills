# Competitive Intel Suite - 竞争情报套件

## 概述

持续监控和分析竞争对手动态，为企业提供竞争情报支持。

## 服务内容包括

- 竞品动态监控
- 产品功能对比
- 价格策略分析
- 营销活动追踪
- 用户反馈分析

## 定价

- 月费：¥699/月
- 支持按竞品数量定制价格

## 使用方法

```bash
competitive-intel-suite --competitors <竞品列表> --metrics <监控指标>
```

## 配置

在 `~/.openclaw/workspace/config/competitive-intel-suite.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 699
}
```
