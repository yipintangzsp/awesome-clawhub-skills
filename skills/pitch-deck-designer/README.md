# pitch-deck-designer

融资 PPT 设计（¥299/次）

## 分类
生产力/融资工具

## 价格
¥299/次

## 安装

```bash
clawhub install pitch-deck-designer
```

## 使用

```bash
pitch-deck-designer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/pitch-deck-designer.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 299,
  "funding_round": "Seed",
  "target_amount": 1000000,
  "industry": "technology"
}
```

## 功能

- 10-15 页标准融资 PPT 结构
- 问题 - 解决方案故事线
- 市场规模分析（TAM/SAM/SOM）
- 商业模式清晰呈现
- 竞争格局可视化
- 财务预测展示
- 团队介绍优化
- 融资需求与用途
- PPT 模板导出（PDF/PPTX）
