## 安装

```bash
clawhub install ai-seo-optimizer
```

## 使用

```bash
ai-seo-optimizer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/ai-seo-optimizer.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 24,
  "target_keywords": ["keyword1", "keyword2"],
  "competitors": ["competitor1.com", "competitor2.com"]
}
```
