# shopify-email-marketing - Shopify 智能邮件营销系统

**痛点**：邮件打开率低，模板设计耗时，客户分层困难，营销效果难追踪
**价值**：AI 生成高转化邮件，自动化客户分层，提升复购率 30-50%

## 使用方式

```bash
shopify-email-marketing --help
```

## 功能特点

- ⚡ **AI 邮件模板生成** - 基于产品自动生成高转化 EDM 模板
- 📊 **客户智能分层** - 基于购买行为自动分组，精准营销
- 💰 **转化率优化** - A/B 测试标题、内容、发送时间，持续优化
- 🔒 **自动化营销流程** - 弃购挽回、新品通知、生日优惠自动触发
- 🎯 **ROI 实时追踪** - 邮件营销效果数据看板，投入产出一目了然

## 适用场景

- Shopify 店铺邮件营销
- 客户复购率提升
- 弃购订单挽回
- 会员生命周期管理

## 配置

在 `~/.openclaw/workspace/config/shopify-email-marketing.json` 配置 SkillPay API Key。
