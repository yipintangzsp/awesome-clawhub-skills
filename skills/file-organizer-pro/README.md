# file-organizer-pro

文件整理专家（SkillPay 收费版）

## 分类
AI/生产力

## 价格
¥29/月

## 安装

```bash
clawhub install file-organizer-pro
```

## 使用

```bash
file-organizer-pro [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/file-organizer-pro.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
