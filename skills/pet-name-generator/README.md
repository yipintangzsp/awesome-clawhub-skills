# pet-name-generator

宠物名字生成（¥2/次）

## 分类
娱乐休闲

## 价格
¥2/次

## 安装

```bash
clawhub install pet-name-generator
```

## 使用

```bash
pet-name-generator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/pet-name-generator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 2
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
