# course-outline-creator

课程大纲创建（SkillPay 收费版）

## 分类
内容创作

## 价格
¥39/次

## 安装

```bash
clawhub install course-outline-creator
```

## 使用

```bash
course-outline-creator [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/course-outline-creator.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
