# Xiaohongshu Viral Copy - 小红书爆款文案

## Description
小红书爆款文案生成工具，基于平台热门内容和流量密码生成高互动笔记文案。包含标题、正文、标签、emoji 搭配，提升笔记曝光和转化。

## Usage
```bash
# 生成文案
/xiaohongshu-viral-copy --topic "护肤心得" --type "share" --audience "female_18_30"

# 标题优化
/xiaohongshu-viral-copy --action "title" --input "普通标题"

# 爆款分析
/xiaohongshu-viral-copy --action "analyze" --note-id "xxx"
```

### 参数说明
- `--topic`: 笔记主题
- `--type`: 笔记类型 (share, review, tutorial, unboxing, before-after)
- `--audience`: 目标受众 (female_18_30, male, student, working)
- `--action`: 操作类型 (write, title, analyze, optimize)
- `--input`: 输入内容
