# cross-chain-bridge-monitor - 跨链桥安全监控系统

**痛点**：跨链桥黑客事件频发，流动性不足导致资金被困，桥接风险难以评估
**价值**：7x24 小时实时监控，提前预警风险，保障跨链资产安全

## 使用方式

```bash
cross-chain-bridge-monitor --help
```

## 功能特点

- ⚡ **多链桥监控** - 支持 Wormhole/LayerZero/Stargate 等主流跨链桥
- 📊 **流动性分析** - 实时监控各链流动性池深度，预警流动性不足
- 💰 **费率对比优化** - 自动对比各桥费率，选择最优跨链方案
- 🔒 **安全风险预警** - 监控异常交易，预警潜在黑客攻击
- 🎯 **桥接历史分析** - 分析各桥历史安全记录，评估可信度

## 适用场景

- 跨链转账前风险评估
- 多链资产配置管理
- 大额跨链资金安全监控
- 跨链套利机会发现

## 配置

在 `~/.openclaw/workspace/config/cross-chain-bridge-monitor.json` 配置 SkillPay API Key。
