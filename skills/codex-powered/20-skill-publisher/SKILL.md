# Skill Publisher (Codex-Powered)

**AI Skill 发布器 | ClawHub 发布 | 版本管理 | 自动打包 | SkillPay 收费**

## 功能
- 自动打包 Skill 发布
- 生成发布说明
- 上传到 ClawHub
- 版本管理

## 收费模式
- **按次收费**: ¥29/次
- **包月订阅**: ¥299/月

## 使用方法
```bash
/skill skill-publisher ./my-skill --version 1.0.0
/skill skill-publisher "发布到 ClawHub" --skill ./skill-path --public
```
