# Code Generator (Codex-Powered)

**AI 代码生成器 | 多语言代码自动生成 | 函数/类/模块完整实现 | SkillPay 收费**

## 功能
- 根据自然语言描述生成完整代码
- 支持 Python、JavaScript、TypeScript、Go、Rust 等语言
- 生成可运行的生产级代码
- 自动添加注释和文档字符串

## 收费模式
- **按次收费**: ¥9/次 (基础函数/类)
- **包月订阅**: ¥99/月 (无限生成)

## 使用方法
```bash
/skill code-generator "创建一个快速排序算法，Python 实现"
/skill code-generator "生成一个 React  hooks 组件，带类型定义" --lang typescript
```

## 环境变量
- `CODEX_API_KEY`: OpenAI Codex API 密钥
- `SKILLPAY_API_KEY`: SkillPay 收费接口密钥

## 输出
- 完整可运行的代码文件
- 使用示例
- 复杂度分析
