# obsidian-second-brain - AI 第二大脑核心技能

## 描述
基于 Obsidian 的个人知识管理系统，实现笔记自动链接、双向引用、知识图谱可视化。支持智能标签、MOC（Map of Content）自动生成、每日笔记模板。

## 定价
**¥299/月** - 高级知识管理订阅

## 功能特性
- 📝 自动双向链接生成
- 🗂️ MOC 地图自动创建
- 🏷️ 智能标签推荐
- 📊 知识图谱可视化
- 📅 每日笔记模板
- 🔗 跨笔记引用追踪

## 用法
```bash
# 安装技能
openclaw skills install obsidian-second-brain

# 配置 Obsidian 仓库路径
openclaw obsidian config --vault ~/ObsidianVault

# 启用自动整理
openclaw obsidian auto-link --enable

# 生成知识图谱
openclaw obsidian graph --output graph.png
```
