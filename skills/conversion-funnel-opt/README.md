# Conversion Funnel Opt - 转化漏斗优化

## 概述

分析用户转化漏斗，识别流失环节，提供优化建议提升转化率。

## 服务内容包括

- 漏斗搭建
- 流失分析
- 优化建议
- A/B 测试
- 效果追踪

## 定价

- 单次：¥699/次
- 支持按漏斗复杂度定制价格

## 使用方法

```bash
conversion-funnel-opt --funnel <漏斗阶段> --data <数据来源>
```

## 配置

在 `~/.openclaw/workspace/config/conversion-funnel-opt.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_analysis": 699
}
```
