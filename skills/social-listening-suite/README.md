# Social Listening Suite - 社交监听套件

## 概述

监控社交媒体上的品牌提及和舆情，实时掌握用户声音和市场趋势。

## 服务内容包括

- 品牌提及监控
- 情感分析
- 竞品对比
- 趋势发现
- 危机预警

## 定价

- 月费：¥599/月
- 支持按监听范围定制价格

## 使用方法

```bash
social-listening-suite --keywords <监听关键词> --platforms <社交平台>
```

## 配置

在 `~/.openclaw/workspace/config/social-listening-suite.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
