# password-manager-ai

密码管理器 AI（SkillPay 收费版）

## 分类
AI/生产力

## 价格
¥19/月

## 安装

```bash
clawhub install password-manager-ai
```

## 使用

```bash
password-manager-ai [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/password-manager-ai.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
