# ai-market-research

AI 市场调研（¥199/月）

## 分类
AI/市场调研

## 价格
¥199/月

## 安装

```bash
clawhub install ai-market-research
```

## 使用

```bash
ai-market-research [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/ai-market-research.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 19,
  "industry": "Technology",
  "target_market": "China",
  "research_depth": "comprehensive"
}
```

## 功能

- 市场规模分析
- 竞品研究报告
- 消费者洞察
- 趋势预测
- SkillPay 集成
