# PR Crisis Manager - PR 危机管理

## 概述

为企业提供品牌危机公关处理方案，快速响应和化解公关危机。

## 服务内容包括

- 危机预警监测
- 应对策略制定
- 声明稿撰写
- 媒体沟通
- 舆情修复

## 定价

- 单次：¥999/次
- 支持按危机级别定制价格

## 使用方法

```bash
pr-crisis-manager --crisis-type <危机类型> --severity <严重程度>
```

## 配置

在 `~/.openclaw/workspace/config/pr-crisis-manager.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_crisis": 999
}
```
