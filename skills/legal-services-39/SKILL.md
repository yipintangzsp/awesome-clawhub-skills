# 债权债务管理与催收辅助系统

## 技能描述
债权债务管理工具，借条管理、催收流程、法律文书、诉讼指导、执行追踪，帮助债权人合法有效收回欠款。

## 定价
- **个人版**: ¥399/月
- **企业版**: ¥999/月
- **机构版**: ¥1999/月

## 核心功能
1. 债权信息管理
2. 催收流程指导
3. 催收文书生成
4. 诉讼材料准备
5. 执行申请辅助
6. 时效管理

## 使用方式
```bash
/debt-manager activate
/debt-manager register --debtor info --amount 50000 --due-date 2024-06-01
/debt-manager collect --step 1 --method letter
/debt-manager lawsuit --prepare true
```

## 适用场景
- 个人借贷
- 企业应收账款
- 小额贷款公司
- 律师事务所
