# AI SEO Writer - AI SEO 文章生成

## Description
专业 SEO 文章生成工具，基于关键词研究自动生成高排名文章。包含标题优化、元描述、内链建议、关键词密度控制，帮助内容快速获得搜索排名。

## Usage
```bash
# 生成 SEO 文章
/ai-seo-writer --keyword "AI 写作工具" --length 2000 --intent "informational"

# 优化已有文章
/ai-seo-writer --action "optimize" --input "article.md" --keyword "目标关键词"

# 批量生成
/ai-seo-writer --keywords "keywords.txt" --template "blog"
```

### 参数说明
- `--keyword`: 目标关键词
- `--length`: 文章字数
- `--intent`: 搜索意图 (informational, commercial, transactional, navigational)
- `--action`: 操作类型 (write, optimize, audit)
- `--template`: 文章模板 (blog, product, landing, guide)
