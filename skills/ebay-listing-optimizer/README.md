
## 使用

```bash
ebay-listing-optimizer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/ebay-listing-optimizer.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 69,
  "ebay_api_key": "your_ebay_api_key",
  "marketplace": "US",
  "category": "electronics"
}
```
