# DeFi Treasury Management - DeFi 财库管理

## 概述

为 DAO 和企业提供去中心化财库资产管理服务，最大化资金收益。

## 服务内容包括

- 多链资产监控
- 收益策略优化
- 风险评估管理
- 自动再投资
- 财库报告生成

## 定价

- 月费：¥799/月
- 支持按管理资产规模定制价格

## 使用方法

```bash
defi-treasury-management --treasury <财库名称> --strategy <策略>
```

## 配置

在 `~/.openclaw/workspace/config/defi-treasury-management.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 799
}
```
