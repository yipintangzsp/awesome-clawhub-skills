# cross-border-tax-calculator - 跨境电商智能税务计算器

**痛点**：各国税务规则复杂，VAT 计算容易出错，合规风险高，罚款损失大
**价值**：自动核算多国税务，合规申报提醒，避免高额罚款和封店风险

## 使用方式

```bash
cross-border-tax-calculator --help
```

## 功能特点

- ⚡ **多国 VAT 自动核算** - 支持欧美日等 30+ 国家 VAT 税率自动计算
- 📊 **税务合规提醒** - 申报截止日期提醒，避免逾期罚款
- 💰 **税务筹划建议** - 合法税务优化方案，降低整体税负
- 🔒 **风险预警系统** - 识别税务风险，预警潜在合规问题
- 🎯 **申报文档生成** - 自动生成税务申报所需文档和报表

## 适用场景

- 跨境电商 VAT 申报
- 多国税务合规管理
- 跨境税务筹划
- 税务风险评估

## 配置

在 `~/.openclaw/workspace/config/cross-border-tax-calculator.json` 配置 SkillPay API Key。
