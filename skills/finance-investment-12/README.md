# 加密货币投资组合管理与自动 rebalance 系统

## 产品概述

一站式加密货币资产管理平台，整合所有交易所和钱包资产，自动执行投资策略，优化收益，降低风险。

## 核心功能

### 资产追踪
- 支持 50+ 交易所
- 多链钱包监控
- 实时资产估值
- PnL 分析

### 自动 rebalance
- 目标配置设定
- 阈值触发 rebalance
- 税务优化执行
- 费用最小化

### DCA 定投
- 定期自动买入
- 价格区间策略
- 智能加仓
- 成本平均

### 风险管理
- 止盈止损设置
- 波动率预警
- 相关性分析
- 风险敞口监控

### 税务合规
- 交易记录自动导入
- 成本 basis 计算
- 税务报表生成
- 多国税法支持

## 支持平台
- 币安、OKX、火币、Coinbase
- MetaMask、Ledger、Trust Wallet
- Ethereum、Bitcoin、Solana 等主流链
