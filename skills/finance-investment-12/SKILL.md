# 加密货币投资组合管理与自动 rebalance 系统

## 技能描述
专业加密货币投资组合管理工具，多交易所资产追踪、自动 rebalance、DCA 定投、止盈止损、税务计算、DeFi 收益追踪，一站式加密资产管理。

## 定价
- **标准版**: ¥799/月（5 交易所/10 钱包）
- **专业版**: ¥1599/月（20 交易所/50 钱包）
- **机构版**: ¥2999/月（无限 +API 交易）

## 核心功能
1. 多交易所资产统一视图
2. 自动 rebalance 策略执行
3. DCA 定投自动化
4. 止盈止损自动触发
5. 税务报表自动生成
6. DeFi 质押收益追踪

## 使用方式
```bash
/crypto-portfolio activate
/crypto-portfolio connect --exchange binance
/crypto-portfolio rebalance --strategy target-allocation
/crypto-portfolio dca --set BTC,ETH --amount 1000
```
