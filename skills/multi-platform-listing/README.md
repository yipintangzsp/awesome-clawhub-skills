# Multi Platform Listing - 多平台 Listing

## 概述

实现产品 Listing 一键发布到多个电商平台，自动优化和同步更新。

## 服务内容包括

- Listing 内容优化
- 多平台一键发布
- 价格同步更新
- 库存自动同步
- 销售效果追踪

## 定价

- 月费：¥599/月
- 支持按 Listing 数量定制价格

## 使用方法

```bash
multi-platform-listing --product <产品信息> --platforms <目标平台>
```

## 配置

在 `~/.openclaw/workspace/config/multi-platform-listing.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
