# 社交媒体多平台自动发布系统

## 技能描述
一站式社交媒体管理工具，支持微信公众号、微博、抖音、小红书、B 站、知乎等多平台内容自动发布、定时推送、数据分析、评论管理。

## 定价
- **标准版**: ¥599/月（5 平台）
- **专业版**: ¥1299/月（15 平台）
- **企业版**: ¥2499/月（无限平台 + 团队协作）

## 核心功能
1. 多平台一键发布
2. 智能定时和最佳发布时间推荐
3. 内容自动适配各平台格式
4. 评论和私信统一管理
5. 数据分析和 ROI 追踪
6. AI 内容生成和优化

## 使用方式
```bash
/social-automation activate
/social-automation schedule --content post.md --platforms all
/social-automation analytics --period week
/social-automation engage --auto-reply true
```

## 适用场景
- 品牌社交媒体运营
- 自媒体多平台分发
- 电商产品推广
- KOL/KOC内容管理
