# AI Compliance Checker - AI 合规检查

## 概述

确保 AI 系统符合相关法律法规和行业标准，降低合规风险。

## 服务内容包括

- 数据安全法合规
- 个人信息保护法
- 行业特定法规
- 算法备案指导
- 合规文档模板

## 定价

- 月费：¥499/月
- 支持按合规项目定制价格

## 使用方法

```bash
ai-compliance-checker --regulation <法规类型> --industry <行业>
```

## 配置

在 `~/.openclaw/workspace/config/ai-compliance-checker.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 499
}
```
