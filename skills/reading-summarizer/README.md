## 安装

```bash
clawhub install reading-summarizer
```

## 使用

```bash
reading-summarizer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/reading-summarizer.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 5
}
```
