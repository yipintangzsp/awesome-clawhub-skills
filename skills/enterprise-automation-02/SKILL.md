# 客户关系管理 (CRM) 自动化跟进系统

## 技能描述
智能 CRM 自动化跟进系统，自动追踪客户互动、智能提醒跟进时间、生成个性化跟进内容、分析客户健康度。支持多渠道客户数据整合，提升销售转化率和客户留存率。

## 核心功能
1. 客户互动自动追踪（邮件/电话/会议/微信）
2. 智能跟进时间预测和提醒
3. 个性化跟进内容生成
4. 客户健康度评分和分析
5. 销售 pipeline 自动更新
6. 流失风险预警

## 使用方式
```bash
# 激活技能
/crm-automation activate

# 导入客户数据
/crm-automation import --source salesforce

# 查看待跟进客户
/crm-automation followups --due today

# 生成客户健康报告
/crm-automation health-report --segment vip
```

## 适用场景
- B2B 销售团队客户跟进管理
- 客户成功团队留存管理
- 销售线索培育和转化
- 大客户关系统一管理

## 技术实现
- 集成主流 CRM 系统（Salesforce、HubSpot、纷享销客）
- AI 预测最佳跟进时间
- 自然语言生成个性化跟进内容
- 客户行为分析和健康度建模
