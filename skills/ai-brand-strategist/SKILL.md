# ai-brand-strategist - AI 智能品牌策略顾问系统

**痛点**：品牌定位模糊，视觉识别不统一，传播策略混乱，品牌建设无从下手
**价值**：AI 专业品牌策略规划，系统化品牌建设，提升品牌价值

## 使用方式

```bash
ai-brand-strategist --help
```

## 功能特点

- ⚡ **品牌定位分析** - 基于市场和竞品分析，找到差异化定位
- 📊 **视觉识别建议** - AI 生成 Logo、配色、字体等视觉元素建议
- 💰 **品牌传播规划** - 制定多渠道品牌传播策略和节奏
- 🔒 **品牌健康度评估** - 多维度评估品牌认知、美誉度、忠诚度
- 🎯 **竞品品牌对标** - 分析竞品品牌策略，找出超越机会

## 适用场景

- 新品牌从 0 到 1 建设
- 老品牌升级重塑
- 品牌传播策略制定
- 品牌资产管理

## 配置

在 `~/.openclaw/workspace/config/ai-brand-strategist.json` 配置 SkillPay API Key。
