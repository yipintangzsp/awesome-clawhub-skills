# ai-brand-strategist

AI 品牌策略（¥249/月）

## 分类
AI/品牌

## 价格
¥249/月

## 安装

```bash
clawhub install ai-brand-strategist
```

## 使用

```bash
ai-brand-strategist [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/ai-brand-strategist.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 24,
  "brand_stage": "growth",
  "target_audience": "millennials",
  "brand_values": ["innovation", "sustainability"]
}
```

## 功能

- 品牌定位分析
- 品牌策略规划
- 视觉识别建议
- 品牌传播方案
- SkillPay 集成
