# 交通事故理赔计算与处理系统

## 技能描述
交通事故处理工具，责任认定分析、赔偿项目计算、理赔流程指导、文书生成、律师推荐，帮助受害者获得合理赔偿。

## 定价
- **标准版**: ¥299/月
- **专业版**: ¥699/月
- **律所版**: ¥1499/月

## 核心功能
1. 事故责任分析
2. 赔偿项目计算
3. 理赔流程指导
4. 文书模板生成
5. 保险公司对接
6. 诉讼支持

## 使用方式
```bash
/traffic-accident activate
/traffic-accident calculate --injury minor --city Beijing --age 35
/traffic-accident liability --scenario rear-end
/traffic-accident claim --insurance-company PICC
```

## 适用场景
- 交通事故受害者
- 车主
- 交通事故律师
- 保险理赔员
