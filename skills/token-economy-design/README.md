# Token Economy Design - 代币经济设计

## 概述

提供专业的代币经济学模型设计服务，确保项目代币的长期可持续发展。

## 服务内容包括

- 代币经济模型设计
- 通证分配方案
- 激励机制设计
- 通缩/通胀模型
- 治理机制设计

## 定价

- 单次：¥1999/次
- 支持按项目复杂度定制价格

## 使用方法

```bash
token-economy-design --project <项目名称> --type <代币类型>
```

## 配置

在 `~/.openclaw/workspace/config/token-economy-design.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_project": 1999
}
```
