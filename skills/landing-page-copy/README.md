# landing-page-copy

落地页文案（SkillPay 收费版）

## 分类
内容创作

## 价格
¥59/次

## 安装

```bash
clawhub install landing-page-copy
```

## 使用

```bash
landing-page-copy [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/landing-page-copy.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 9
}
```

## 功能

- 自动收费验证
- SkillPay 集成
- 错误处理
- 详细日志
