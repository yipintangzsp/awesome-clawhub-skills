# Medical Report Interpreter - 医疗报告解读

**专业医疗报告解读服务** - 基于权威医学数据库分析检验指标，提供风险评估和就医建议。

⚠️ **重要声明**：本服务仅供参考，不能替代专业医疗建议、诊断或治疗。

---

## 📋 使用方式

```bash
# 启动服务
medical-report

# 按提示输入报告数据或直接粘贴检验单内容
```

| 服务类型 | 价格 | 说明 |
|----------|------|------|
| 基础解读 | ¥5/次 | 血常规/尿常规等基础检查 |

---

## ✅ 我们能做的

- 🔬 **指标解读** - 解释各项检验指标的临床意义
- 📊 **参考范围** - 提供标准参考值及偏差分析
- ⚠️ **风险提示** - 识别异常指标的潜在风险
- 🏥 **就医引导** - 根据异常程度建议就诊科室

## ❌ 我们不能做的

- ❌ 不能下诊断结论
- ❌ 不能开药方/治疗方案
- ❌ 不能替代医生面诊
- ❌ 不能处理急症/重症

---

## 📝 支持报告类型

| 类别 | 指标示例 |
|------|----------|
| 血常规 | WBC、RBC、HGB、PLT |
| 肝功能 | ALT、AST、TBIL、ALB |
| 肾功能 | CREA、UA、BUN |
| 血糖血脂 | GLU、TC、TG、LDL、HDL |
| 甲状腺功能 | TSH、FT3、FT4、TPOAb |
| 肿瘤标志物 | AFP、CEA、CA199、PSA |

---

## 🚨 紧急情况提示

**如有以下症状，请立即就医，不要等待解读：**

- 🔴 胸痛/呼吸困难
- 🔴 严重外伤/出血
- 🔴 意识模糊/昏迷
- 🔴 剧烈腹痛
- 🔴 其他紧急症状

**拨打 120 或前往最近医院急诊科**

1. 本解读基于用户提供的数据，可能存在信息不完整的情况
2. 检验结果需结合临床症状、病史、其他检查综合判断
3. 本服务不能替代执业医师的诊断和治疗建议
4. 如因依赖本解读而延误治疗，本服务不承担法律责任
5. 最终诊断请以医院执业医师意见为准
