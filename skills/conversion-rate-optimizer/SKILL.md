# conversion-rate-optimizer - 独立站转化率智能优化系统

**痛点**：流量多转化少，不知道问题在哪，盲目优化没效果，广告费打水漂
**价值**：数据驱动诊断转化漏斗，精准定位问题，提升转化率 50-100%

## 使用方式

```bash
conversion-rate-optimizer --help
```

## 功能特点

- ⚡ **购物漏斗诊断** - 分析访客 - 加购 - 支付全流程，定位流失环节
- 📊 **A/B 测试建议** - 基于数据生成可执行的 A/B 测试方案
- 💰 **用户行为分析** - 热力图、点击图、滚动深度深度分析
- 🔒 **竞品对标分析** - 对比行业标杆，找出差距和优化方向
- 🎯 **优化优先级排序** - 基于影响力和实施难度排序优化项

## 适用场景

- Shopify/独立站转化率提升
- 落地页优化
- 结账流程优化
- 广告投放 ROI 提升

## 配置

在 `~/.openclaw/workspace/config/conversion-rate-optimizer.json` 配置 SkillPay API Key。
