## 安装

```bash
clawhub install conversion-rate-optimizer
```

## 使用

```bash
conversion-rate-optimizer [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/conversion-rate-optimizer.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 19,
  "website_url": "https://your-store.com",
  "focus_areas": ["checkout", "product_page", "landing_page"]
}
```

- 转化率分析
