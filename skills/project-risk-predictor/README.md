# Project Risk Predictor - 项目风险预测

## 概述

利用 AI 技术预测项目风险，提前识别潜在问题并提供缓解建议。

## 服务内容包括

- 风险识别
- 概率评估
- 影响分析
- 缓解策略
- 持续监控

## 定价

- 月费：¥599/月
- 支持按项目数量定制价格

## 使用方法

```bash
project-risk-predictor --project <项目信息> --history <历史数据>
```

## 配置

在 `~/.openclaw/workspace/config/project-risk-predictor.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
