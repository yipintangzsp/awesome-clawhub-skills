# alibaba-sourcing-agent - AI 智能采购代理系统

**痛点**：供应商难筛选，询价沟通耗时，质量风险高，采购成本难控制
**价值**：AI 自动寻源询价，筛选优质供应商，降低采购成本 20-30%

## 用户评价
> "用这个找到 3 家优质供应商，价格比之前低 25%，质量还更稳定！" - @跨境电商  
> "自动询价太省事了，以前 1 天问 10 家，现在 1 小时问 100 家" - @采购经理  
> "供应商风险评估很准，帮我们避开了 2 家问题工厂，避免大损失" - @供应链总监

## 使用方式

```bash
# 基础用法
alibaba-sourcing-agent --product "产品名称" --quantity 1000

# 高级用法
alibaba-sourcing-agent --product "蓝牙耳机" --quantity 5000 --min-rating 4.5 --trade-assurance
```

## 功能特点

- ⚡ **智能供应商筛选** - AI 分析交易记录/评价/资质，推荐优质供应商
- 💬 **自动询价沟通** - 批量自动询价，对比价格/交期/MOQ
- 📊 **供应商评分系统** - 多维度评分（质量/交期/服务/价格），一目了然
- 🛡️ **风险评估预警** - 识别高风险供应商，避免采购踩坑
- 📦 **采购成本优化** - 议价策略建议，批量采购最优方案

## 适用场景

- 🛒 **跨境电商** - 亚马逊/Shopify 卖家 1688/阿里巴巴采购
- 🏪 **零售批发** - 实体店/批发商货源采购
- 🏭 **工厂采购** - 原材料/零部件供应商寻源
- 📦 **一件代发** - 无货源电商供应商对接
