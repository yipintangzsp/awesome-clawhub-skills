# etsy-print-on-demand

Etsy 按需印刷（¥59/月）

## 分类
电商/按需印刷

## 价格
¥59/月

## 安装

```bash
clawhub install etsy-print-on-demand
```

## 使用

```bash
etsy-print-on-demand [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/etsy-print-on-demand.json`
2. 添加配置:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 59,
  "pod_platforms": ["printful", "printify"],
  "product_types": ["tshirt", "mug", "poster"],
  "target_margin": 40
}
```
