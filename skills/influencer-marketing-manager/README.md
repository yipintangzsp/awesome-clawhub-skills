# influencer-marketing-manager

网红营销管理（¥179/月）

## 分类
电商/网红营销

## 价格
¥179/月

## 安装

```bash
clawhub install influencer-marketing-manager
```

## 使用

```bash
influencer-marketing-manager [选项]
```

## 配置

1. 创建配置文件 `~/.openclaw/workspace/config/influencer-marketing-manager.json`
2. 添加 SkillPay API Key:
```json
{
  "skillpay_api_key": "your_api_key_here",
  "price_per_call": 17,
  "platforms": ["Instagram", "YouTube", "TikTok"],
  "niche": "Beauty & Fashion",
  "budget_range": "1000-5000"
}
```

## 功能

- 网红资源库
- 合作管理
- ROI 追踪
- 效果分析
- SkillPay 集成
