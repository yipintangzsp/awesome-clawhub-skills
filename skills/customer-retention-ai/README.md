# Customer Retention AI - 客户留存 AI

## 概述

利用 AI 技术预测客户流失风险，制定个性化留存策略，提升客户生命周期价值。

## 服务内容包括

- 流失风险预测
- 客户分群
- 个性化触达
- 留存活动
- 效果分析

## 定价

- 月费：¥599/月
- 支持按客户规模定制价格

## 使用方法

```bash
customer-retention-ai --segment <客户分群> --channels <触达渠道>
```

## 配置

在 `~/.openclaw/workspace/config/customer-retention-ai.json` 中配置：

```json
{
  "skillpay_api_key": "your_api_key",
  "price_per_month": 599
}
```
