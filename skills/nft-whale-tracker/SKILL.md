# nft-whale-tracker - NFT 巨鲸实时追踪系统

**痛点**：错过 NFT 大戶动向，无法及时发现优质项目，信息滞后导致高位接盘
**价值**：实时监控巨鲸钱包，提前发现价值项目，跟随聪明钱获取超额收益

## 使用方式

```bash
nft-whale-tracker --help
```

## 功能特点

- ⚡ **巨鲸钱包监控** - 追踪 Top 1000 NFT 收藏家和大戶实时交易
- 📊 **链上数据分析** - 深度分析交易模式，识别洗盘和拉盘行为
- 💰 **异动实时提醒** - 大额买入/卖出即时推送，不错过任何机会
- 🔒 **项目风险评估** - 分析持仓集中度，预警 Rug Pull 风险
- 🎯 **聪明钱跟随** - 识别高胜率交易者，一键复制成功策略

## 适用场景

- NFT 投资前尽职调查
- 发现早期价值项目
- 监控持仓大户动向
- 识别市场操纵行为

## 配置

在 `~/.openclaw/workspace/config/nft-whale-tracker.json` 配置 SkillPay API Key。
