# dao-treasury-analyzer - DAO 财库健康度分析系统

**痛点**：DAO 财库数据分散，难以评估项目财务健康，投资决策缺乏依据
**价值**：一站式财库分析，透明化资金流向，科学评估 DAO 投资价值

## 使用方式

```bash
dao-treasury-analyzer --help
```

## 功能特点

- ⚡ **财库健康度评分** - 基于储备、支出、收入的多维度评估体系
- 📊 **资金流向追踪** - 实时监控大额转账，识别异常资金流动
- 💰 **收入来源分析** - 分解协议收入、投资收益、捐赠等各来源占比
- 🔒 **治理参与度分析** - 投票率、提案质量、社区活跃度综合评估
- 🎯 **投资价值评级** - 基于财库数据给出买入/持有/卖出建议

## 适用场景

- DAO 代币投资前调研
- 治理代币价值评估
- 协议收入追踪分析
- DeFi 项目尽职调查

## 配置

在 `~/.openclaw/workspace/config/dao-treasury-analyzer.json` 配置 SkillPay API Key。
