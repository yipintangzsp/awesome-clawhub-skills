# API Doc Generator

API 文档生成工具 - 从代码注释自动生成 API 文档

## 使用方式

```bash
api-doc-generator --input src/routes.js --format markdown
```

## 功能特点

- 📝 自动解析注释
- 📄 多格式输出
- 🔗 交互式文档
- 💰 ¥19/月订阅

## 配置

在 `~/.openclaw/workspace/config/api-doc-generator.json` 配置 SkillPay API Key。
