# 另类投资数据分析系统（私募/VC/PE）

## 技能描述
私募股权、风险投资、对冲基金等另类投资数据分析工具，基金业绩追踪、GP 评估、项目估值、退出分析，服务专业投资者和机构。

## 定价
- **专业版**: ¥1999/月
- **机构版**: ¥2999/月

## 核心功能
1. 私募基金业绩数据库
2. GP 能力和信誉评估
3. 项目估值和回报分析
4. 退出案例研究
5. 行业趋势分析
6. LP 配置建议

## 使用方式
```bash
/alt-investment activate
/alt-investment search --strategy tech-vc --vintage 2020
/alt-investment gp-evaluate --name Sequoia
/alt-investment benchmark --fund FUND123
```

## 适用场景
- 家族办公室
- 母基金 (FOF)
- 保险公司投资部
- 高净值个人
