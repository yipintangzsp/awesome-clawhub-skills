# defi-yield-aggregator - DeFi 收益聚合优化器

**痛点**：DeFi 协议太多难以选择，手动切换收益低，Gas 费高昂，错过最佳收益时机
**价值**：自动对比 100+ 协议 APY，智能切换最优收益，年化提升 40-80%

## 使用方式

```bash
defi-yield-aggregator --help
```

## 功能特点

- ⚡ **多协议收益对比** - 实时监控 Aave/Compound/Curve 等 100+ 协议 APY
- 📊 **智能复投策略** - 自动复利计算，最优复投频率，最大化复合收益
- 💰 **Gas 费优化** - 智能选择低 Gas 时段执行，节省交易成本 30%+
- 🔒 **风险评估** - 协议安全评分，TVL 监控，预警潜在风险
- 🎯 **一键收益优化** - 自动将资金切换到当前最高收益协议

## 适用场景

- DeFi 流动性挖矿收益最大化
- 多链收益聚合管理
- 稳定币理财自动优化
- 机构级 DeFi 资产配置

## 配置

在 `~/.openclaw/workspace/config/defi-yield-aggregator.json` 配置 SkillPay API Key。
