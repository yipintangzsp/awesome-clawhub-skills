# 🎨 封面图 + 标题优化指南

**核心原则**：3 秒法则 - 用户 3 秒内决定是否点击

---

## 📐 封面图设计规范

### 尺寸要求
- **知乎专栏**：900×500px (16:9)
- **Twitter/X**：1200×675px (16:9)
- **Medium**：1400×788px (16:9)
- **小红书**：1080×1440px (3:4)

### 设计要素

**必须包含**：
1. ✅ 醒目主标题（不超过 10 个字）
2. ✅ 高对比度配色
3. ✅ 相关视觉元素
4. ✅ 品牌标识（ClawHub logo）

**避免**：
1. ❌ 文字过多（超过 20 个字）
2. ❌ 低分辨率图片
3. ❌ 颜色超过 3 种
4. ❌ 复杂背景

---

## 🎯 爆款标题公式

### 公式 1：数字 + 结果 + 时间
```
"3 步做到 XXX，仅需 5 分钟"
"7 天涨粉 10000 的秘诀"
```

### 公式 2：警告/危险 + 解决方案
```
"⚠️ 90% 的人都在犯的错，这样改"
"别再 XXX 了！正确方法是..."
```

### 公式 3：对比 + 具体数字
```
"从 0 到 100 万，我只做了这 3 件事"
"同样发笔记，为什么她流量是我的 100 倍"
```

### 公式 4：疑问 + 利益
```
"你的钱包值$5000？30 秒查空投"
"还在手动 XXX？这个工具 1 秒搞定"
```

### 公式 5：独家/内部 + 稀缺
```
"内部流出的 XXX 方法，删前速看"
"仅限前 100 名，XXX 工具 5 折优惠"
```

## 🎨 配色方案

### 警示类（币圈安全）
- 主色：红色 #FF0000
- 辅色：黑色 #000000、黄色 #FFFF00
- 情绪：紧迫、危险、注意

### 财富类（空投/赚钱）
- 主色：绿色 #00FF00、金色 #FFD700
- 辅色：深蓝 #003366、白色 #FFFFFF
- 情绪：诱惑、专业、信任

### 增长类（流量/运营）
- 主色：粉色 #FF69B4、橙色 #FFA500
- 辅色：白色 #FFFFFF、浅灰 #F0F0F0
- 情绪：活泼、积极、向上

### 工具类（效率/办公）
- 主色：蓝色 #0066FF、青色 #00CCCC
- 辅色：白色 #FFFFFF、深灰 #333333
- 情绪：专业、可靠、高效

## 📊 A/B 测试建议

**测试要素**：
1. 标题文案（同图不同标题）
2. 封面图片（同标题不同图）
3. 发布时间（早/中/晚）
4. 价格锚点（原价 vs 折扣价）

**测试周期**：每个版本 3-7 天
**成功标准**：点击率提升 20%+

## 📝 检查清单

发布前确认：
- [ ] 标题不超过 20 个字
- [ ] 封面图清晰无模糊
- [ ] 主标题在手机上可读
- [ ] 配色对比度足够
- [ ] 有明确行动指令
- [ ] 包含数字或具体利益
- [ ] 无错别字
- [ ] 链接可点击

## 🎯 案例参考

### 优秀案例
```
标题："⚠️ 90% 新币 7 天归零！3 步自救指南"
封面：红色警告背景 + 大字号"90% 归零" + 加密货币图标
结果：点击率 15%+
```

### 失败案例
```
标题："关于加密货币投资的一些建议和注意事项"
封面：模糊的比特币图片 + 小字号文字
结果：点击率 2%-
```
