### 2026-03-07: OpenClaw 工具权限配置
**问题**：2026 年 3 月 2 日后部署的实例默认关闭工具权限
**解决**：
```json
{
  "tools": {
    "profile": "full",
    "exec": {
      "security": "full",
      "ask": "off"
    },
    "sessions": {
      "visibility": "all"
    }
  }
}
```
**标签**：#openclaw #配置 #权限

### 2026-03-07: SkillPay Skill 发布频率限制
**问题**：ClawHub 限制每小时最多发布 5 个新 Skill
**解决**：批量创建文件，分批发布，每小时 5 个
**标签**：#clawhub #发布 #限制
